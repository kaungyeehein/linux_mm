### နိဒါန်း (Introduction)
Linux တွင် အရာအားလုံးနီးပါးကို "File" အဖြစ် သတ်မှတ်သည်။ Configuration များ၊ စာသားများသာမက hardware device များသည်လည်း file များပင် ဖြစ်သည်။ ထို file များကို စနစ်တကျ စုစည်းရန် Directory (Windows တွင် Folder ဟု ခေါ်သည်) များကို အသုံးပြုသည်။ ဤသင်ခန်းစာတွင် Linux file system အတွင်း မည်သို့သွားလာရမည်နှင့် file များကို မည်သို့ကြည့်ရှုရမည်ကို လေ့လာပါမည်။

---

### Files and Directories (ဖိုင်များနှင့် ဒိုင်ရက်ထရီများ)
Linux သည် **Hierarchical Directory Structure** (အဆင့်ဆင့် ချိတ်ဆက်ထားသော ပုံစံ) ကို အသုံးပြုသည်။ ၎င်းသည် ပြောင်းပြန်လှန်ထားသော သစ်ပင်တစ်ပင်နှင့် တူပြီး အမြစ် (Root) မှ စတင်သည်။

* **Root Directory (`/`):** System ၏ အမြင့်ဆုံး အဆင့်ဖြစ်သည်။ အခြား file များနှင့် directory အားလုံးသည် ၎င်းအောက်တွင် ရှိသည်။
* **Home Directory (`~`):** အသုံးပြုသူတစ်ဦးစီအတွက် ကိုယ်ပိုင် နေရာဖြစ်သည်။ ဥပမာ- `/home/bob`။



---

### Absolute vs. Relative Paths
File တစ်ခု သို့မဟုတ် Directory တစ်ခု၏ တည်နေရာကို ဖော်ပြရန် နည်းလမ်းနှစ်မျိုးရှိသည်-

1.  **Absolute Path (အပြည့်အစုံ လမ်းကြောင်း):** Root (`/`) မှ စတင်၍ လမ်းကြောင်းတစ်ခုလုံးကို ဖော်ပြခြင်းဖြစ်သည်။ ၎င်းသည် သင် မည်သည့်နေရာသို့ ရောက်နေသည်ဖြစ်စေ အမြဲတမ်း တူညီသည်။
    * ဥပမာ- `/home/user/Documents/file.txt`
2.  **Relative Path (နှိုင်းယှဉ် လမ်းကြောင်း):** သင် လက်ရှိရောက်ရှိနေသော နေရာ (Current Working Directory) မှ စတင်၍ ဖော်ပြခြင်းဖြစ်သည်။
    * ဥပမာ- သင်သည် `/home/user` ထဲတွင် ရောက်နေပါက `Documents/file.txt` ဟု ရေးရုံဖြင့် လုံလောက်သည်။

**အထူးအမှတ်အသားများ:**
* **`.` (အစက်တစ်စက်):** လက်ရှိရောက်နေသော directory ကို ဆိုလိုသည်။
* **`..` (အစက်နှစ်စက်):** လက်ရှိနေရာ၏ အပေါ်က directory (Parent directory) ကို ဆိုလိုသည်။

---

### Listing Files with `ls` (ဖိုင်များကို စာရင်းပြုစု ကြည့်ရှုခြင်း)
`ls` command သည် directory တစ်ခုအတွင်းရှိ အရာများကို ကြည့်ရန် အသုံးအများဆုံး ဖြစ်သည်။

* **`ls` (ရိုးရိုး):** ဖိုင်အမည်များကိုသာ ပြသည်။
* **`ls -l` (Long format):** ဖိုင်၏ ခွင့်ပြုချက် (permissions)၊ ပိုင်ရှင် (owner)၊ အရွယ်အစားနှင့် နောက်ဆုံးပြင်ဆင်ခဲ့သော အချိန်တို့ကို အသေးစိတ်ပြသည်။
* **`ls -a` (All files):** ဖျောက်ထားသော file (Hidden files) များအပါအဝင် အားလုံးကို ပြသည်။
* **`ls -h` (Human Readable):** ဖိုင်အရွယ်အစားကို KB, MB, GB များဖြင့် ဖတ်ရလွယ်အောင် ပြသည်။ (များသောအားဖြင့် `ls -lh` ဟု တွဲသုံးသည်)
* **`ls -R` (Recursive):** လက်ရှိ directory သာမက ၎င်းအောက်ရှိ directory အခွဲများထဲမှ file များကိုပါ အကုန်ပြသည်။



---

### Hidden Files (ဖျောက်ထားသော ဖိုင်များ)
Linux တွင် file အမည် သို့မဟုတ် directory အမည်၏ ရှေ့ဆုံး၌ **အစက် (`.`)** ပါရှိပါက ၎င်းကို hidden file/directory ဟု သတ်မှတ်သည်။ ၎င်းတို့ကို ပုံမှန် `ls` ဖြင့် မမြင်ရဘဲ `ls -a` သုံးမှသာ မြင်ရမည်ဖြစ်သည်။ များသောအားဖြင့် configuration file များ (ဥပမာ- `.bashrc`) အတွက် အသုံးပြုသည်။

---

### Navigating Directories with `cd` (လမ်းကြောင်း ပြောင်းလဲခြင်း)
`cd` (Change Directory) command ကို လမ်းကြောင်း ပြောင်းရန် သုံးသည်။

* **`cd /etc`:** Absolute path သုံး၍ `/etc` သို့ သွားခြင်း။
* **`cd ..`:** အပေါ်က directory သို့ ပြန်ထွက်ခြင်း။
* **`cd ~` သို့မဟုတ် `cd`:** မိမိ၏ home directory သို့ ချက်ချင်း ပြန်သွားခြင်း။
* **`cd -`:** ယခင် နောက်ဆုံး ရောက်ခဲ့သော directory သို့ ပြန်သွားခြင်း။

---

### Useful Commands for Directory Info
* **`pwd` (Print Working Directory):** သင် လက်ရှိ ဘယ်နေရာ ရောက်နေသလဲ ဆိုသည်ကို အပြည့်အစုံ ပြသည်။
* **`tree`:** Directory ဖွဲ့စည်းပုံကို သစ်ပင်ပုံစံ (Tree format) ဖြင့် မြင်သာအောင် ပြသည်။ (System တိုင်းတွင် default မပါဝင်နိုင်ပါ)

---

### အကျဉ်းချုပ် (Summary)
ဤသင်ခန်းစာတွင် သင်ယူခဲ့ရသည်များမှာ-
* Linux File System ၏ အဆင့်ဆင့် ဖွဲ့စည်းပုံ (Root, Home)။
* Absolute နှင့် Relative paths များ၏ ကွာခြားချက်နှင့် `.` ၊ `..` တို့၏ အဓိပ္ပာယ်။
* `ls` command ၏ option ပေါင်းစုံဖြင့် file များကို အသေးစိတ်ကြည့်ရှုပုံ။
* Hidden files များ၏ သဘောသဘာဝ။
* `cd` နှင့် `pwd` ကို သုံး၍ directory များအတွင်း သွားလာပုံ။
