### နိဒါန်း (Introduction)
သင်သည် ကွန်ပျူတာစနစ်ကို အိမ်မှာဖြစ်စေ၊ တက္ကသိုလ်မှာဖြစ်စေ သို့မဟုတ် လုပ်ငန်းခွင်တစ်ခုမှာဖြစ်စေ အသုံးပြုနေသည်ဖြစ်ပါစေ၊ သင်အသုံးပြုမည့် operating system (OS) အပေါ် ဆုံးဖြတ်ချက်တစ်ခု ချမှတ်ရမည်ဖြစ်သည်။ ဤဆုံးဖြတ်ချက်ကို သင်ကိုယ်တိုင် ချမှတ်နိုင်သလို (အထူးသဖြင့် သင့်ကိုယ်ပိုင်ကွန်ပျူတာဖြစ်ပါက)၊ သင့်လုပ်ငန်းခွင်ရှိ စနစ်များအားလုံးအတွက် ရွေးချယ်မှုပြုလုပ်ရန်လည်း တာဝန်ရှိနိုင်ပါသည်။ အမြဲတမ်းလိုပင်၊ ရရှိနိုင်သော ရွေးချယ်စရာများအကြောင်းကို သေချာစွာ သိရှိထားခြင်းက တာဝန်ယူမှုရှိသော ဆုံးဖြတ်ချက်တစ်ခု ချမှတ်ရာတွင် အထောက်အကူပြုပါလိမ့်မည်။ ဤသင်ခန်းစာတွင် သင်စဉ်းစားနေနိုင်သည့် operating system ရွေးချယ်စရာများအကြောင်းကို သိရှိစေရန် ရည်ရွယ်ပါသည်။

### Operating System ဆိုတာဘာလဲ (What is an Operating System)
Operating system တစ်ခုကို မရွေးချယ်မီ ကျွန်ုပ်တို့ သေချာစွာ သိထားရမည့် အချက်တစ်ခုမှာ ဤဝေါဟာရ၏ အဓိပ္ပာယ်ပင်ဖြစ်သည်။ Operating system သည် သင့်ကွန်ပျူတာ၏ နှလုံးသားသဖွယ်ဖြစ်ပြီး ၎င်း၏အတွင်း၌ သို့မဟုတ် ၎င်း၏အပေါ်တွင် အသုံးချပရိုဂရမ်များ (applications) ကို အလုပ်လုပ်စေနိုင်ပါသည်။ ထို့အပြင် operating system တွင် disks များ၊ partitions များ၊ မျက်နှာပြင်များ (screens)၊ ကီးဘုတ်များ၊ network cards စသည့် ကွန်ပျူတာ hardware များအား အသုံးပြုနိုင်ရန် drivers များ ပါဝင်မည်ဖြစ်သည်။ ကျွန်ုပ်တို့သည် operating system ကို အတိုကောက်အားဖြင့် OS ဟု ခေါ်ဆိုလေ့ရှိသည်။

ယနေ့ခေတ်တွင် စီးပွားရေးလုပ်ငန်းသုံး ကွန်ပျူတာများအတွက်ရော အိမ်သုံးအတွက်ပါ ရရှိနိုင်သော operating system များစွာရှိပါသည်။ ကျွန်ုပ်တို့သည် ရရှိနိုင်သော ရွေးချယ်မှုများကို ရိုးရှင်းစေရန် အောက်ပါအတိုင်း အုပ်စုဖွဲ့နိုင်ပါသည် -

* Linux အခြေခံ Operating Systems များ
    * လုပ်ငန်းသုံး (Enterprise) Linux
    * လူသုံးကုန် (Consumer) Linux
* UNIX
* macOS
* Windows အခြေခံ Operating Systems များ
    * Windows Servers
    * Windows Desktops

### Linux Distribution တစ်ခုကို ရွေးချယ်ခြင်း (Choosing a Linux Distribution)

#### Linux Kernel နှင့် Linux Distributions များ
Linux distributions များအကြောင်း ပြောဆိုသည့်အခါ operating system မှာ Linux ဖြစ်သည်။ Linux သည် kernel ဖြစ်ပြီး Linux distribution တိုင်း၏ အဓိက ဗဟိုချက်ဖြစ်သည်။ Linux kernel ၏ software ကို Linus Torvalds ဦးဆောင်သော လူပုဂ္ဂိုလ်တစ်စုမှ ထိန်းသိမ်းကိုင်တွယ်သည်။ Torvalds သည် Linux kernel အပေါ် အလုပ်လုပ်ရန် The Linux Foundation ဟုခေါ်သော စက်မှုလုပ်ငန်းဆိုင်ရာ အဖွဲ့အစည်းတစ်ခုတွင် ခန့်အပ်ခြင်းခံရသည်။

> **မှတ်ချက် (NOTE):** Linux kernel ကို ဖင်လန်နိုင်ငံမှ ကျောင်းသားတစ်ဦးဖြစ်သူ Linus Torvalds က ၁၉၉၁ ခုနှစ်တွင် စတင်ထုတ်လုပ်ခဲ့သည်။ ၁၉၉၂ ခုနှစ်တွင် GNU General Public License version 2 (GPLv2) အောက်၌ ပထမဆုံး ထုတ်ဝေခဲ့သော Kernel ဗားရှင်းမှာ 0.12 ဖြစ်သည်။

#### Linux Kernel
အထက်တွင် ဖော်ပြခဲ့သည့်အတိုင်း Linux distribution အားလုံးသည် တူညီသော operating system ဖြစ်သည့် Linux အပေါ်တွင် အလုပ်လုပ်ကြသည်။

#### Linux Distribution
လူများက Red Hat Linux သို့မဟုတ် Ubuntu Linux ဟု ပြောဆိုကြသောအခါ ၎င်းတို့သည် Linux distribution (ဖြန့်ချိမှု) ကို ရည်ညွှန်းခြင်းဖြစ်သည်။ Linux distribution တွင် Linux kernel နှင့်အတူ ၎င်းကို ကျွန်ုပ်တို့ အပြန်အလှန် အသုံးပြုနိုင်စေမည့် ပတ်ဝန်းကျင် (environment) တစ်ခု ပါဝင်မည်ဖြစ်သည်။ အနည်းဆုံးအားဖြင့် ကျွန်ုပ်တို့သည် စနစ်ကို ဝင်ရောက်အသုံးပြုရန်နှင့် စီမံခန့်ခွဲရန်အတွက် Bash ကဲ့သို့သော command line shell တစ်ခုနှင့် အခြေခံ commands များ အစုံအလင် လိုအပ်မည်ဖြစ်သည်။ များသောအားဖြင့် Linux distribution များတွင် Gnome သို့မဟုတ် KDE ကဲ့သို့သော Desktop Environment အပြည့်အစုံ ပါဝင်လေ့ရှိသည်။

Linux distribution တစ်ခုစီသည် Linux operating system ကို အသုံးပြုသော်လည်း၊ ၎င်းတို့ အသုံးပြုသော operating system ၏ ဗားရှင်းမှာ ကွာခြားနိုင်ပါသည်။ ဆိုလိုသည်မှာ distribution တစ်ခု စတင်ပွင့်လာချိန် (boots) တွင် အသုံးပြုနေသော Linux Kernel ဗားရှင်းကို ဆိုလိုခြင်းဖြစ်သည်။

> **အကြံပြုချက် (TIP):** အကယ်၍ သင့်တွင် လက်ရှိတွင် Linux command line ကို အသုံးပြုနိုင်ပါက၊ kernel release ကို ဖတ်ခြင်းဖြင့် သင်အသုံးပြုနေသော Linux kernel ဗားရှင်းကို အလွယ်တကူ စစ်ဆေးနိုင်ပါသည် -
>
> $ uname -r
> 4.15.0-1019-aws

#### Linux Distribution အမျိုးအစားများ (Types of Linux Distributions)
Linux kernel ၏ နောက်ဆုံးထွက်ဗားရှင်းကို အမြဲတမ်း အသုံးပြုခြင်းက အကောင်းဆုံးဟု ထင်ရသော်လည်း အမှန်တကယ်တွင် ထိုမျှမရိုးရှင်းပါ။ ကျွန်ုပ်တို့သည် Linux distributions များကို အကြမ်းဖျင်းအားဖြင့် အုပ်စုသုံးစု ခွဲခြားနိုင်ပါသည် -

* **လုပ်ငန်းသုံးအဆင့် (Enterprise Grade) Linux Distributions များ**
    * Red Hat Enterprise Linux
    * CentOS
    * SUSE Linux Enterprise Server
    * Debian GNU/Linux
    * Ubuntu LTS
* **လူသုံးကုန်အဆင့် (Consumer Grade) Linux Distributions များ**
    * Fedora
