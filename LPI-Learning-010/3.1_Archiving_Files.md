### နိဒါန်း (Introduction)

ဤသင်ခန်းစာသည် Linux command line ကို အသုံးပြု၍ ဖိုင်များနှင့် directory များကို မည်သို့ archive ပြုလုပ်ရမည် (ပေါင်းစည်းသိမ်းဆည်းရမည်) နှင့် compress ပြုလုပ်ရမည် (အရွယ်အစားချုံ့ရမည်) ကို လေ့လာကြမည်ဖြစ်သည်။ နေ့စဉ် လုပ်ငန်းဆောင်တာများတွင် backup ပြုလုပ်ရန် သို့မဟုတ် ဖိုင်များစွာကို တစ်ခုတည်းအဖြစ် ပေါင်းစည်း၍ ပေးပို့ရန်အတွက် Archiving နှင့် Compression သည် အလွန်အရေးကြီးသော ကျွမ်းကျင်မှုတစ်ခု ဖြစ်သည်။

### Tar Utility (The Tape Archiver)

`tar` command သည် Linux တွင် ဖိုင်များကို archive ပြုလုပ်ရန် အသုံးအများဆုံး utility ဖြစ်သည်။ ၎င်းကို မူလက tape backup များအတွက် ရည်ရွယ်ခဲ့သော်လည်း ယခုအခါ ဖိုင်များစွာကို archive file (.tar) တစ်ခုတည်းအဖြစ် စုစည်းရန် အသုံးပြုသည်။

အသုံးများသော `tar` option များမှာ -
* `-c`: Archive အသစ်တစ်ခု ဖန်တီးရန် (Create)
* `-v`: လုပ်ဆောင်ချက်များကို အသေးစိတ်ပြသရန် (Verbose)
* `-f`: Archive ဖိုင်အမည်ကို သတ်မှတ်ရန် (File)
* `-x`: Archive ထဲမှ ဖိုင်များကို ပြန်ထုတ်ရန် (Extract)
* `-t`: Archive ထဲရှိ ဖိုင်စာရင်းကို ကြည့်ရန် (List)

### Compression (ဖိုင်အရွယ်အစား ချုံ့ခြင်း)

Archive ပြုလုပ်ထားသော ဖိုင်များကို သိမ်းဆည်းရန်နေရာ သက်သာစေရန်အတွက် compression tools များကို အသုံးပြုကြသည်။ Linux တွင် အောက်ပါ tool များကို အသုံးများသည်။

၁။ **gzip**: အသုံးအများဆုံးဖြစ်ပြီး `.gz` extension ကို အသုံးပြုသည်။ `tar` နှင့် တွဲဖက်လျှင် `.tar.gz` သို့မဟုတ် `.tgz` ဖြစ်လာသည်။
၂။ **bzip2**: gzip ထက် ပိုမိုချုံ့နိုင်သော်လည်း အချိန်ပိုကြာတတ်သည်။ `.bz2` extension ကို အသုံးပြုသည်။
၃။ **xz**: အကောင်းဆုံး ချုံ့နိုင်စွမ်းရှိသည်။ `.xz` extension ကို အသုံးပြုသည်။

---

### Compression Tools (ဖိုင်အရွယ်အစား ချုံ့သည့် ကိရိယာများ)

ဖိုင်များကို archive ပြုလုပ်ရုံသာမက ၎င်းတို့၏ အရွယ်အစားကို သေးငယ်သွားအောင် ချုံ့ရန် (Compression) tool များကို အသုံးပြုကြသည်။ Linux တွင် အသုံးအများဆုံး tools များမှာ `gzip`, `bzip2` နှင့် `xz` တို့ ဖြစ်ကြသည်။


* **gzip**: အသုံးအများဆုံးဖြစ်ပြီး မြန်ဆန်သည်။ ချုံ့ထားသော ဖိုင်များသည် `.gz` extension ဖြင့် အဆုံးသတ်သည်။
* **bzip2**: `gzip` ထက် ပိုမိုချုံ့နိုင်သော်လည်း လုပ်ဆောင်ချက် အနည်းငယ် နှေးကွေးနိုင်သည်။ `.bz2` extension ကို အသုံးပြုသည်။
* **xz**: ဤကိရိယာသည် အကောင်းဆုံး ချုံ့နိုင်စွမ်း (highest compression ratio) ကို ပေးစွမ်းသည်။ `.xz` extension ကို အသုံးပြုသည်။

### Archiving Tools (ဖိုင်များကို ပေါင်းစည်းသိမ်းဆည်းသည့် ကိရိယာများ)

`tar` command သည် Linux တွင် ဖိုင်များနှင့် directory များစွာကို archive ဖိုင်တစ်ခုတည်းအဖြစ် ပေါင်းစည်းရန် အခြေခံအကျဆုံး tool ဖြစ်သည်။

#### Tar ၏ အသုံးများသော Option များ:
* `-c` (Create): Archive အသစ်တစ်ခု ဖန်တီးရန်။
* `-x` (Extract): Archive ထဲရှိ ဖိုင်များကို ပြန်ထုတ်ရန်။
* `-v` (Verbose): လုပ်ဆောင်နေသော အခြေအနေများကို အသေးစိတ်ပြသရန်။
* `-f` (File): Archive ဖိုင်၏ အမည်ကို သတ်မှတ်ရန်။
* `-j`: `bzip2` ကို အသုံးပြု၍ ချုံ့ရန်။
* `-z`: `gzip` ကို အသုံးပြု၍ ချုံ့ရန်။
* `-J`: `xz` ကို အသုံးပြု၍ ချုံ့ရန်။

---

### Managing ZIP Files (ZIP ဖိုင်များကို စီမံခန့်ခွဲခြင်း)

Windows စနစ်များနှင့် ဖိုင်ဖလှယ်ရာတွင် `zip` နှင့် `unzip` command များကို အသုံးပြုသည်။
* `zip archive_name.zip file1 file2`: ဖိုင်များကို zip archive အဖြစ် ဖန်တီးရန်။
* `unzip archive_name.zip`: zip archive မှ ဖိုင်များကို ပြန်ထုတ်ရန်။

၁။ `tar` ကို အသုံးပြု၍ `documents` ဆိုသော directory ကို `backup.tar.gz` အမည်ဖြင့် `gzip` ချုံ့နည်းသုံးကာ archive ပြုလုပ်ပါ။
၂။ `archive.tar` ထဲတွင် မည်သည့်ဖိုင်များ ပါဝင်နေသည်ကို extract မလုပ်ဘဲ မည်သို့ကြည့်ရှုမည်နည်း။
၃။ `.tar.bz2` ဖိုင်တစ်ခုကို ပြန်ထုတ် (extract) ရန် မည်သည့် option များကို အသုံးပြုရမည်နည်း။

# ၃.၁ Command Line ပေါ်တွင် ဖိုင်များကို သိမ်းဆည်းခြင်း (Archiving) - အပိုင်း (၂)

### Guided Exercises (လမ်းညွှန်လေ့ကျင့်ခန်းများ)

၁။ အောက်ပါ command၊ option(s) နှင့် argument(s) အပိုင်းများကို ခွဲခြားပြပါ- 
   * `tar -czf backup.tar.gz /home/user/Documents`
     * **Command:** `tar` 
     * **Option:** `-czf` 
     * **Argument:** `backup.tar.gz`, `/home/user/Documents` 

၂။ `documents` ဆိုသော directory ကို `backup.tar.gz` အမည်ဖြင့် `gzip` ချုံ့နည်းသုံးကာ archive ပြုလုပ်ရန် command ကို ရေးသားပါ။ 
   * **အဖြေ:** `tar -czf backup.tar.gz documents` 

၃။ `archive.tar` ဖိုင်ထဲတွင် မည်သည့်ဖိုင်များ ပါဝင်နေသည်ကို extract မလုပ်ဘဲ ကြည့်ရှုရန် မည်သည့် option ကို အသုံးပြုရမည်နည်း။ 
   * **အဖြေ:** `-t` (သို့မဟုတ် `--list`) option ကို အသုံးပြုရပါမည်။ (ဥပမာ- `tar -tf archive.tar`)

### Explorational Exercises (ရှာဖွေလေ့လာခြင်း လေ့ကျင့်ခန်းများ)

၁။ `tar` command တွင် `-c` နှင့် `-x` option နှစ်ခုလုံးကို command တစ်ခုတည်း၌ တစ်ပြိုင်နက် အသုံးပြု၍ ရပါသလား။ အဘယ်ကြောင့်နည်း။ 

၂။ သင်၏ home directory ထဲရှိ ဖိုင်အချို့ကို `zip` command သုံး၍ ချုံ့ကြည့်ပါ။ ထို့နောက် `unzip` ကို အသုံးပြု၍ ၎င်းဖိုင်များကို directory အသစ်တစ်ခုထဲသို့ ပြန်ထုတ်ကြည့်ပါ။ 

၃။ `gzip`, `bzip2` နှင့် `xz` တို့ဖြင့် ချုံ့ထားသော ဖိုင်များကို မည်သို့ခွဲခြားနိုင်သနည်း။ ၎င်းတို့၏ extension များမှာ အဘယ်နည်း။

### Summary (အနှစ်ချုပ်)

ဤသင်ခန်းစာတွင် သင်ယူခဲ့ရသည်များမှာ- 
* **Archiving**: ဖိုင်များနှင့် directory များကို `tar` အသုံးပြု၍ archive ဖိုင်တစ်ခုတည်းအဖြစ် ပေါင်းစည်းနည်း။ 
* **Compression**: သိမ်းဆည်းသည့်နေရာ သက်သာစေရန် `gzip`, `bzip2` နှင့် `xz` တို့ကို အသုံးပြု၍ ဖိုင်အရွယ်အစား ချုံ့နည်း။ 
* **ZIP files**: Windows စနစ်များနှင့် လိုက်ဖက်ညီသော `zip` နှင့် `unzip` command များ အသုံးပြုပုံ။ 

**အသုံးပြုခဲ့သော Command များ:** 
* `tar`: Archive များ ဖန်တီးရန်၊ ကြည့်ရန်နှင့် ပြန်ထုတ်ရန်။ 
* `gzip` / `gunzip`: `.gz` ဖိုင်များကို ချုံ့ရန်နှင့် ပြန်ဖြည်ရန်။ 
* `bzip2` / `bunzip2`: `.bz2` ဖိုင်များကို ချုံ့ရန်နှင့် ပြန်ဖြည်ရန်။ 
* `xz` / `unxz`: `.xz` ဖိုင်များကို ချုံ့ရန်နှင့် ပြန်ဖြည်ရန်။ 
* `zip` / `unzip`: `.zip` ဖိုင်များကို စီမံခန့်ခွဲရန်။
