### နိဒါန်း (Introduction)
Linux operating system တွင် အလုပ်လုပ်သည့်အခါ ဖိုင်စနစ် (filesystem) အတွင်း မည်သို့သွားလာရမည်နှင့် ဖိုင်များကို မည်သို့ရှာဖွေရမည်ကို သိရှိရန် အလွန်အရေးကြီးပါသည်။ ဤသင်ခန်းစာတွင် Directory များ၏ တည်ဆောက်ပုံ၊ ဖိုင်အမည်ပေးစနစ်များ၊ နှင့် လက်ရှိရောက်ရှိနေသောနေရာကို သိရှိစေရန် သို့မဟုတ် အခြားနေရာသို့ ပြောင်းလဲသွားလာရန် အသုံးပြုသော command များအကြောင်းကို လေ့လာရမည်ဖြစ်ပါသည်။

### ဖိုင်များနှင့် Directory များ (Files and Directories)
Linux တွင် အချက်အလက်များကို သိမ်းဆည်းရန်အတွက် "Files" များကို အသုံးပြုပြီး ထိုဖိုင်များကို စနစ်တကျ စုစည်းသိမ်းဆည်းရန် "Directories" (Windows တွင် Folder ဟု ခေါ်ဆိုသော အရာများ) ကို အသုံးပြုပါသည်။ Linux ဖိုင်စနစ်သည် အပင်တစ်ပင်၏ အမြစ်ကဲ့သို့ (Tree structure) တည်ဆောက်ထားပြီး အပေါ်ဆုံးအဆင့်ကို Root Directory `/` ဟု ခေါ်ဆိုပါသည်။
```bash
$ tree
Documents
├── Mission-Statement.txt └── Reports
└── report2018.txt 1 directory, 2 files
 ```

### ဖိုင်နှင့် Directory အမည်များ (File and Directory Names)
Linux တွင် ဖိုင် သို့မဟုတ် Directory အမည်ပေးသည့်အခါ အောက်ပါအချက်များကို သတိပြုရပါမည် -
* **Case Sensitivity:** Linux သည် စာလုံးအကြီးအသေးကို ခွဲခြားပါသည်။ ဥပမာ - `File.txt` နှင့် `file.txt` သည် မတူညီသော ဖိုင်နှစ်ခုဖြစ်ပါသည်။
* **Special Characters:** အမည်များတွင် စာလုံးများ၊ ဂဏန်းများ၊ အစက် (dot)၊ underscore (_) နှင့် dash (-) တို့ကို အသုံးပြုသင့်ပါသည်။ space (ကွက်လပ်) သို့မဟုတ် အခြားအထူးသင်္ကေတများကို ရှောင်ကြဉ်သင့်ပါသည်။
* **Hidden Files:** အစက် `.` နှင့် စတင်သော အမည်ရှိသည့် ဖိုင်များကို ဖျောက်ထားသော ဖိုင်များ (Hidden Files) ဟု သတ်မှတ်ပါသည်။

### ဖိုင်စနစ်အတွင်း သွားလာခြင်း (Navigating the Filesystem)
Directory များအတွင်း ကူးသန်းသွားလာရန် အောက်ပါ အခြေခံ command များကို အသုံးပြုနိုင်ပါသည်။

1. **`pwd` (Print Working Directory):** မိမိလက်ရှိရောက်ရှိနေသော လမ်းကြောင်းအပြည့်အစုံကို ပြသပေးပါသည်။
```bash
$ pwd
/home/user
```

2. **`cd` (Change Directory):** တစ်နေရာမှ တစ်နေရာသို့ ပြောင်းလဲသွားလာရန် အသုံးပြုပါသည်။
```bash
$ cd /var/log
$ cd ~
```

3. **`ls` (List):** လက်ရှိ Directory အတွင်းရှိ ဖိုင်များနှင့် Directory များ၏ အမည်များကို စာရင်းပြုစုပြသပေးပါသည်။

### Absolute and Relative Paths
ဖိုင်တစ်ခု သို့မဟုတ် Directory တစ်ခု၏ တည်နေရာကို ဖော်ပြရန် နည်းလမ်းနှစ်မျိုးရှိပါသည်။
* Absolute Path: Root `/` မှ စတင်၍ လမ်းကြောင်းအပြည့်အစုံကို ရေးသားခြင်းဖြစ်ပါသည်။ မည်သည့်နေရာမှမဆို အသုံးပြုနိုင်ပါသည်။ ဥပမာ - `/home/user/Documents/work.txt`
* Relative Path: မိမိလက်ရှိရောက်ရှိနေသော နေရာမှ စတင်၍ ဖော်ပြခြင်းဖြစ်ပါသည်။
  * `.` (အစက်တစ်စက်) သည် လက်ရှိ directory ကို ကိုယ်စားပြုသည်။
  * `..` (အစက်နှစ်စက်) သည် လက်ရှိနေရာ၏ အပေါ်အဆင့် (Parent Directory) ကို ကိုယ်စားပြုသည်။ ဥပမာ - လက်ရှိနေရာသည် `/home/user` ဖြစ်ပါက `Documents/work.txt` ဟု ရေးသားခြင်းသည် Relative Path ဖြစ်ပါသည်။

### နိဒါန်း (Introduction)
ဤဒုတိယပိုင်းသင်ခန်းစာတွင် Linux စနစ်အတွင်းရှိ အသုံးပြုသူတစ်ဦးစီ၏ ကိုယ်ပိုင်နေရာ (Home Directory) များအကြောင်း၊ လမ်းကြောင်းများကို အတိုကောက်ရေးသားနည်းများ၊ ဖျောက်ထားသော ဖိုင်များကို ကြည့်ရှုနည်းနှင့် ဖိုင်များ၏ အသေးစိတ်အချက်အလက်များကို စစ်ဆေးနည်းတို့ကို ဆက်လက်လေ့လာသွားပါမည်။ 

### Home Directories (Home Directory များ)
Linux စနစ်ရှိ အသုံးပြုသူတိုင်းတွင် ၎င်းတို့၏ ကိုယ်ပိုင်ဖိုင်များ၊ configuration များ သိမ်းဆည်းရန် သီးသန့် Directory တစ်ခုစီ ရှိကြသည်။ ၎င်းကို Home Directory ဟု ခေါ်ဆိုသည်။ 
* ပုံမှန်အားဖြင့် လမ်းကြောင်းမှာ `/home/username` ဖြစ်သည်။ (ဥပမာ - အသုံးပြုသူအမည် `carol` အတွက် `/home/carol` ဖြစ်သည်)။ 
* စနစ်စီမံခန့်ခွဲသူ (System Administrator) သို့မဟုတ် `root` အသုံးပြုသူ၏ Home Directory မှာမူ `/root` ဖြစ်သည်။ 

### Home အတွက် အထူးနှိုင်းရလမ်းကြောင်း (The Special Relative Path for Home)
Home directory သို့ လျင်မြန်စွာ ရည်ညွှန်းနိုင်ရန် `~` (tilde) သင်္ကေတကို အသုံးပြုသည်။ 
* `~` သင်္ကေတသည် လက်ရှိ login ဝင်ထားသော အသုံးပြုသူ၏ home directory ကို ဆိုလိုသည်။ 
* `cd ~` ဟု ရိုက်နှိပ်ပါက မိမိ၏ home directory သို့ ချက်ချင်းရောက်ရှိမည် ဖြစ်သည်။ 
* `~username` (ဥပမာ - `~carol`) ဟု ရေးသားပါက သတ်မှတ်ထားသော အသုံးပြုသူ၏ home directory လမ်းကြောင်းကို ဆိုလိုသည်။ 

### ဖျောက်ထားသော ဖိုင်များနှင့် Directory များ (Hidden Files and Directories)
Linux တွင် အစက် `.` နှင့် စတင်သော ဖိုင် သို့မဟုတ် Directory များကို hidden အဖြစ် သတ်မှတ်သည်။ ၎င်းတို့ကို ပုံမှန် `ls` command ဖြင့် မမြင်နိုင်ပါ။ 
* ၎င်းတို့ကို ကြည့်ရှုရန် `-a` (all) option ကို အသုံးပြုရမည်။
```bash
$ ls -a
```

### အသေးစိတ်စာရင်းပြုစုခြင်း Option (The Long List Option)
ဖိုင်များ၏ အမည်သာမက အသေးစိတ်အချက်အလက်များ (ဥပမာ - permissions, ပိုင်ရှင်၊ ဖိုင်ဆိုဒ်၊ ပြင်ဆင်ခဲ့သည့်အချိန်) ကို သိရှိလိုပါက `-l` option ကို အသုံးပြုသည်။
```bash
$ ls -l
-rw-r--r-- 1 user staff 3606 Jan 13 2017 report2018.txt
```
* `-rw-r--r--`: ဖိုင်အမျိုးအစား နှင့် permissions
* `1`: ဖိုင်မှာချိတ်ဆက်ထားသော အရေအတွက်
* `user staff`: ဖိုင်ကိုပိုင်သော user name
*​`3606`: ဖိုင်ရဲ့ဆိုဒ် (bytes)
* `Jan 13 2017`: ဖိုင်ကိုနောက်ဆုံးပြင်ဆင်ခဲ့သော အချိန်
* `report2018.txt`: ဖိုင်အမည်

### ထပ်ဆောင်း ls Option များ (Additional ls Options)
* `-h` (Human Readable): ဖိုင်ဆိုဒ်များကို KB, MB, GB စသည်ဖြင့် ဖတ်ရှုရလွယ်ကူအောင် ပြသပေးသည်။ (အများအားဖြင့် `-l` နှင့် တွဲသုံးသည် - `ls -lh`)။ 
* `-R` (Recursive): လက်ရှိ directory သာမက ၎င်းအောက်ရှိ subdirectory များအတွင်းမှ အကြောင်းအရာအားလုံးကိုပါ တစ်ခါတည်း ပြသပေးသည်။

### အကျဉ်းချုပ် (Summary)
ဤသင်ခန်းစာတွင် အောက်ပါအချက်များကို လေ့လာခဲ့ပြီးဖြစ်သည်- 
* Linux filesystem အတွင်း cd နှင့် pwd ကို အသုံးပြု၍ သွားလာခြင်း။
* Home directory အတွက် ~ အား အသုံးပြုပုံ။
* Absolute paths (တိကျသောလမ်းကြောင်း) နှင့် Relative paths (လက်ရှိနေရာမှနှိုင်းရလမ်းကြောင်း) တို့၏ ကွာခြားချက်။
* Hidden files (အစက်နှင့်စသောဖိုင်များ) နှင့် ၎င်းတို့ကို ls -a ဖြင့် ကြည့်ရှုပုံ။
