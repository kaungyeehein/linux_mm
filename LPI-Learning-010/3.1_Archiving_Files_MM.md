# 3.1 Archiving Files on the Command Line

## သင်ခန်းစာ အညွှန်း (Reference to LPI objectives)
* **Linux Essentials version 1.6, Exam 010, Objective 3.1**
* **အလေးချိန် (Weight):** 2

### အဓိက သိရှိရမည့် နယ်ပယ်များ (Key knowledge areas)
* Archiving (ဖိုင်များကို စုစည်းသိမ်းဆည်းခြင်း) နှင့် Compression (ဖိုင်အရွယ်အစား ချုံ့ခြင်း) တို့၏ ကွာခြားချက်
* `tar` command ကို အသုံးပြုခြင်း
* Compression tools များ (`gzip`, `bzip2`, `xz`) ကို အသုံးပြုခြင်း
* ZIP file များကို ကိုင်တွယ်ခြင်း (`zip`, `unzip`)

---

### နိဒါန်း (Introduction)
ဖိုင်အမြောက်အမြားကို သိမ်းဆည်းရန် သို့မဟုတ် တစ်နေရာမှ တစ်နေရာသို့ ပေးပို့ရန်အတွက် ၎င်းတို့ကို ဖိုင်တစ်ခုတည်းအဖြစ် စုစည်းခြင်း (Archiving) နှင့် နေရာလွတ်သက်သာစေရန် ချုံ့ခြင်း (Compression) တို့ကို ပြုလုပ်ရသည်။ Linux တွင် ဤလုပ်ငန်းစဉ်များအတွက် သီးခြား tool များ ရှိပါသည်။

---

### Archiving vs. Compression (စုစည်းခြင်း နှင့် ချုံ့ခြင်း)
* **Archiving:** ဖိုင်များစွာနှင့် directory များကို မူရင်းအတိုင်း ဖိုင်တစ်ခုတည်း (Archive file) အဖြစ် စုစည်းလိုက်ခြင်းဖြစ်သည်။ ၎င်းသည် ဖိုင်အရွယ်အစားကို မလျှော့ချပေးပါ။
* **Compression:** Algorithm များ အသုံးပြု၍ ဖိုင်၏ အရွယ်အစားကို သေးငယ်သွားအောင် ပြုလုပ်ခြင်းဖြစ်သည်။

---

### Managing Archives with `tar`
`tar` (Tape Archive) သည် Linux တွင် အသုံးအများဆုံး archiving tool ဖြစ်သည်။ ၎င်းသည် ဖိုင်များကို စုစည်းပေးရုံသာမက compression tool များနှင့်လည်း တွဲဖက်လုပ်ဆောင်နိုင်သည်။

#### အသုံးများသော Options များ:
* `-c` (Create): Archive အသစ်တစ်ခု တည်ဆောက်ရန်။
* `-v` (Verbose): လုပ်ဆောင်နေသော ဖိုင်များကို စာရင်းပြရန်။
* `-f` (File): Archive ဖိုင်၏ အမည်ကို သတ်မှတ်ရန်။
* `-t` (List): Archive ထဲရှိ ဖိုင်များကို ကြည့်ရန်။
* `-x` (Extract): Archive ထဲမှ ဖိုင်များကို ပြန်ထုတ်ရန်။

#### ဥပမာများ:
* **Archive ပြုလုပ်ခြင်း:** `tar -cvf backup.tar /home/user/docs`
* **Archive ထဲမှ ဖိုင်များကြည့်ခြင်း:** `tar -tvf backup.tar`
* **Archive ပြန်ဖြည်ခြင်း:** `tar -xvf backup.tar`

---

### Compression Tools (ဖိုင်ချုံ့သည့် ကိရိယာများ)
Linux တွင် အဓိက အသုံးပြုသော compression နည်းလမ်း သုံးမျိုး ရှိသည်-

1. **`gzip` (.gz):** အမြန်ဆုံး ဖြစ်သော်လည်း အချုံ့နိုင်စွမ်း အနည်းဆုံးဖြစ်သည်။
   * `gzip file.txt` (file.txt.gz ဖြစ်သွားမည်)
   * `gunzip file.txt.gz` (ပြန်ဖြည်ရန်)
2. **`bzip2` (.bz2):** `gzip` ထက် ပိုချုံ့နိုင်သော်လည်း ပိုကြာသည်။
   * `bzip2 file.txt`
   * `bunzip2 file.txt.bz2`
3. **`xz` (.xz):** အကောင်းဆုံး ချုံ့နိုင်သော်လည်း အချိန်အကြာဆုံးနှင့် resource အသုံးအများဆုံး ဖြစ်သည်။
   * `xz file.txt`
   * `unxz file.txt.xz`

#### `tar` နှင့် Compression တွဲသုံးခြင်း:
* **.tar.gz (z):** `tar -czvf backup.tar.gz /folder`
* **.tar.bz2 (j):** `tar -cjvf backup.tar.bz2 /folder`
* **.tar.xz (J):** `tar -cJvf backup.tar.xz /folder`

---

### ZIP Files
Windows နှင့် အခြား system များတွင် အသုံးများသော `.zip` format ကိုလည်း Linux တွင် အသုံးပြုနိုင်သည်။ ZIP သည် archiving နှင့် compression ကို တစ်ပြိုင်တည်း လုပ်ဆောင်ပေးသည်။

* **Zip ပြုလုပ်ခြင်း:** `zip -r backup.zip /folder` (`-r` သည် directory အောက်ရှိ အရာအားလုံးပါဝင်စေရန်ဖြစ်သည်)
* **Zip ဖြည်ခြင်း:** `unzip backup.zip`

---

### အကျဉ်းချုပ် (Summary)
ယခု သင်ခန်းစာတွင် သင်ယူခဲ့ရသည်များမှာ-
* Archiving နှင့် Compression တို့၏ အခြေခံ သဘောတရားများ။
* `tar` command ကို အသုံးပြု၍ ဖိုင်များကို စုစည်းခြင်း၊ စာရင်းကြည့်ခြင်းနှင့် ပြန်ဖြည်ခြင်း။
* `gzip`, `bzip2` နှင့် `xz` တို့၏ အားသာချက်၊ အားနည်းချက်များနှင့် အသုံးပြုပုံ။
* `zip` နှင့် `unzip` ကို အသုံးပြု၍ common zip archives များကို ကိုင်တွယ်ပုံ။
