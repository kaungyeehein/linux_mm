# ၁.၁ Linux Evolution and Popular Operating Systems (Linux ၏ ဆင့်ကဲပြောင်းလဲမှုနှင့် လူကြိုက်များသော Operating Systems များ)

[span_0](start_span)ဤသင်ခန်းစာသည် LPI Objective 1.1 (Weight: 2) ကို အခြေခံထားပါသည်။[span_0](end_span)

### Introduction (နိဒါန်း)
Linux သည် ကမ္ဘာပေါ်တွင် လူကြိုက်အများဆုံး Operating System များထဲမှ တစ်ခုဖြစ်သည်။ [span_1](start_span)၎င်းကို ၁၉၉၁ ခုနှစ်တွင် Linus Torvalds မှ စတင်ဖန်တီးခဲ့ခြင်းဖြစ်ပြီး ၁၉၇၀ ပြည့်နှစ်များက AT&T Laboratories မှ ထုတ်လုပ်ခဲ့သော Unix OS ကို အတုယူ၍ ရေးသားခဲ့ခြင်းဖြစ်သည်။[span_1](end_span)

[span_2](start_span)Unix သည် ထိုခေတ်အခါက တန်ဖိုးကြီးမားသော စက်ကြီးများအတွက်သာ ရည်ရွယ်ခဲ့သော်လည်း Linus သည် လူသာမန်များသုံးနိုင်သည့် x86 platform (သာမန်ရုံးသုံး ကွန်ပျူတာများ) တွင် အသုံးပြုနိုင်ရန် Unix နှင့် ဆင်တူသော Linux ကို တီထွင်ခဲ့သည်။[span_2](end_span)

* [span_3](start_span)Linux သည် Unix ၏ မူဝါဒများနှင့် စနစ်များကို အသုံးပြုထားသော်လည်း Unix ၏ code များ ပါဝင်ခြင်းမရှိဘဲ သီးခြားလွတ်လပ်သော project တစ်ခုဖြစ်သည်။[span_3](end_span)
* [span_4](start_span)၎င်းကို ကုမ္ပဏီတစ်ခုတည်းမှ ပိုင်ဆိုင်ခြင်းမဟုတ်ဘဲ နိုင်ငံတကာမှ Programmer များ စုပေါင်းထားသော Community မှ ထိန်းသိမ်းလုပ်ဆောင်နေခြင်းဖြစ်သည်။[span_4](end_span)
* [span_5](start_span)Linux သည် အခမဲ့ရရှိနိုင်ပြီး မည်သူမဆို ကန့်သတ်ချက်မရှိဘဲ အသုံးပြုနိုင်သည်။[span_5](end_span)

### Distributions (Linux ဖြန့်ချိမှုပုံစံများ)
[span_6](start_span)Linux Distribution ဆိုသည်မှာ Linux Kernel နှင့် အခြားလိုအပ်သော Application များကို ကုမ္ပဏီတစ်ခု သို့မဟုတ် Community တစ်ခုမှ စုစည်းကာ အသုံးပြုရလွယ်ကူအောင် ပြုလုပ်ထားသော Package တစ်ခုဖြစ်သည်။[span_6](end_span)

**အဓိက Distribution မိသားစုများ:**

၁။ **[span_7](start_span)Debian Family:**[span_7](end_span)
* **[span_8](start_span)Debian GNU/Linux:** အလွန်စိတ်ချရသော OS တစ်ခုဖြစ်ပြီး Ian Murdock မှ ၁၉၉၃ တွင် စတင်ခဲ့သည်။[span_8](end_span)
* **Ubuntu:** Mark Shuttleworth မှ ၂၀၀၄ ခုနှစ်တွင် စတင်ခဲ့ပြီး Desktop ပေါ်တွင် အသုံးပြုရလွယ်ကူစေရန် ရည်ရွယ်သည်။ [span_9](start_span)၆ လတစ်ကြိမ် Version အသစ်ထွက်ရှိပြီး ၂ နှစ်တစ်ကြိမ် LTS (Long Term Support) Version ထွက်ရှိသည်။[span_9](end_span)

၂။ **[span_10](start_span)Red Hat Family:**[span_10](end_span)
* **[span_11](start_span)Red Hat Enterprise Linux (RHEL):** စီးပွားရေးလုပ်ငန်းသုံး Server များအတွက် အဓိကထားပြီး IBM မှ ပိုင်ဆိုင်သည်။[span_11](end_span)
* **[span_12](start_span)CentOS:** RHEL ၏ Source Code ကို အခြေခံထားပြီး အခမဲ့သုံးနိုင်သော်လည်း ကုမ္ပဏီ၏ Support မရရှိပါ။[span_12](end_span)
* **[span_13](start_span)Fedora:** နည်းပညာအသစ်များကို အမြန်ဆုံး စမ်းသပ်အသုံးပြုလေ့ရှိသော Distribution ဖြစ်သည်။[span_13](end_span)

၃။ **[span_14](start_span)SUSE Family:**[span_14](end_span)
* **[span_15](start_span)SUSE Linux Enterprise (SLES):** ဂျာမနီနိုင်ငံတွင် စတင်ခဲ့ပြီး YaST ဟုခေါ်သော configuration tool ကြောင့် လူသိများသည်။[span_15](end_span)
* **[span_16](start_span)openSUSE:** Developer များနှင့် User များ စမ်းသပ်သုံးစွဲနိုင်ရန် အခမဲ့ပေးထားသော Version ဖြစ်သည်။[span_16](end_span)

### Embedded Systems (အထူးပြုစနစ်များ)
[span_17](start_span)Embedded Systems ဆိုသည်မှာ ကိရိယာတစ်ခုခု (ဥပမာ- ကား၊ ဆေးဘက်ဆိုင်ရာစက်ပစ္စည်း) အတွင်း အထူးလုပ်ဆောင်ချက်တစ်ခုလုပ်ရန် ထည့်သွင်းထားသော Hardware နှင့် Software ပေါင်းစပ်မှုဖြစ်သည်။[span_17](end_span)

**ထင်ရှားသော Embedded Linux များ:**
* **Android:** Google မှ တီထွင်ထားပြီး Touchscreen ဖုန်းများ၊ TV နှင့် နာရီများတွင် အသုံးပြုသည်။ [span_18](start_span)Linux Kernel ကို အခြေခံထားသည်။[span_18](end_span)
* **[span_19](start_span)Raspbian (Raspberry Pi OS):** Raspberry Pi ဟုခေါ်သော ကတ်ပြားအရွယ် ကွန်ပျူတာလေးများတွင် အသုံးပြုရန် Debian ကို အခြေခံပြီး ပြုလုပ်ထားခြင်းဖြစ်သည်။[span_19](end_span)

### Linux and the Cloud (Linux နှင့် Cloud စနစ်)
[span_20](start_span)Cloud Computing ဆိုသည်မှာ ကွန်ပျူတာအရင်းအမြစ်များကို အင်တာနက်ပေါ်မှတစ်ဆင့် ဝယ်ယူ သို့မဟုတ် ငှားရမ်းအသုံးပြုခြင်းဖြစ်သည်။[span_20](end_span)

* [span_21](start_span)Public Cloud များ၏ ၉၀% သည် Linux ပေါ်တွင် အလုပ်လုပ်နေသည်။[span_21](end_span)
* [span_22](start_span)Amazon Web Services (AWS), Google Cloud Platform (GCP) နှင့် Microsoft Azure တို့သည် Linux ကို အခြေခံထားသော Virtual Machines (IaaS) များကို ဝန်ဆောင်မှုပေးကြသည်။[span_22](end_span)

### Definitions (အဓိပ္ပာယ်ဖွင့်ဆိုချက်များ)
* **[span_23](start_span)Kernel:** Operating System တစ်ခု၏ အဓိကအကျဆုံး အစိတ်အပိုင်းဖြစ်ပြီး Hardware နှင့် Software ကြား ဆက်သွယ်ပေးသည်။[span_23](end_span)
* **Package Manager:** Software များကို အလွယ်တကူ Install လုပ်ရန်၊ [span_24](start_span)[span_25](start_span)Update လုပ်ရန်နှင့် ဖျက်ပစ်ရန် အသုံးပြုသော စနစ် (ဥပမာ- `dpkg`, `rpm`)။[span_24](end_span)[span_25](end_span)
* **[span_26](start_span)LTS (Long Term Support):** နှစ်ရှည်လုံခြုံရေး Update များ ပေးမည့် Version ပုံစံ။[span_26](end_span)
