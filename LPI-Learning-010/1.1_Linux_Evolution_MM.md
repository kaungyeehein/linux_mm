1.1 Linux Evolution and Popular Operating Systems
နိဒါန်း (Introduction)
Linux သည် ကမ္ဘာပေါ်တွင် အသုံးအများဆုံး Operating Systems များထဲမှ တစ်ခုဖြစ်သည်။ ၎င်းကို ၁၉၉၁ ခုနှစ်တွင် Linus Torvalds မှ စတင်ဖန်တီးခဲ့ခြင်းဖြစ်သည်။ Linux သည် ၁၉၇၀ ပြည့်နှစ်များက AT&T Laboratories မှ ထုတ်လုပ်ခဲ့သော Unix Operating System အပေါ် အခြေခံကာ စိတ်ကူးရယူ တည်ဆောက်ခဲ့ခြင်း ဖြစ်သည်။ ထိုခေတ်အခါက Unix သည် ခေတ်မီပြီး အဆင့်မြင့်သော်လည်း x86 platform သုံး ရုံးသုံးကွန်ပျူတာများတွင် အလွယ်တကူ အသုံးပြုနိုင်ခြင်း မရှိသေးပေ။ ထို့ကြောင့် ကျောင်းသားတစ်ဦးဖြစ်သော Linus သည် ဤ platform ပေါ်တွင် run နိုင်မည့် Unix-like Operating System တစ်ခုကို စတင်အကောင်အထည်ဖော်ခဲ့ခြင်း ဖြစ်သည်။
Linux သည် Unix ၏ အခြေခံသဘောတရားများကို အသုံးပြုထားသော်လည်း Unix ၏ code များကို တိုက်ရိုက်အသုံးပြုထားခြင်း မဟုတ်ဘဲ သီးခြားလွတ်လပ်သော project တစ်ခု ဖြစ်သည်။ ၎င်းကို ကုမ္ပဏီတစ်ခုတည်းမှ ပိုင်ဆိုင်ခြင်းမျိုး မဟုတ်ဘဲ နိုင်ငံတကာမှ programmer များ စုပေါင်းထားသော community မှ ထိန်းသိမ်းလုပ်ဆောင်နေခြင်း ဖြစ်သည်။ Linux သည် လွတ်လပ်စွာ ရရှိနိုင်သော (freely available) အရာဖြစ်သောကြောင့် မည်သူမဆို ကန့်သတ်ချက်မရှိဘဲ အသုံးပြုနိုင်သည်။
Distributions
Linux Distribution (သို့မဟုတ် Distro) ဆိုသည်မှာ Linux Kernel နှင့် အသုံးပြုသူများအတွက် လိုအပ်သော applications များကို စုစည်းထားသော package တစ်ခုဖြစ်သည်။ ကုမ္ပဏီများ သို့မဟုတ် user community များက ၎င်းတို့ကို သတ်မှတ်ထားသော အသုံးပြုသူအုပ်စု သို့မဟုတ် လုပ်ဆောင်ချက်များအတွက် အကောင်းဆုံးဖြစ်အောင် ပြုပြင်မွမ်းမံထားကြသည်။
Distributions များကို ၎င်းတို့၏ မိသားစုအုပ်စု (Family) အလိုက် အောက်ပါအတိုင်း ခွဲခြားနိုင်သည် -
1. Debian Family
ဤမိသားစုဝင် Distro များသည် software များကို စီမံခန့်ခွဲရန် dpkg package manager ကို အသုံးပြုကြသည်။
 * Debian GNU/Linux: ၁၉၉၃ ခုနှစ်တွင် Ian Murdock မှ စတင်ခဲ့ပြီး အလွန်ခိုင်မာစိတ်ချရသော (reliable) OS တစ်ခုဖြစ်သည်။ ၎င်းသည် user များ၏ လွတ်လပ်ခွင့်ကို လေးစားသောကြောင့် proprietary software များကို default အနေဖြင့် ထည့်သွင်းမထားပေ။
 * Ubuntu: ၂၀၀၄ ခုနှစ်တွင် Mark Shuttleworth မှ စတင်ခဲ့ပြီး Desktop ပတ်ဝန်းကျင်တွင် အသုံးပြုရလွယ်ကူစေရန် ရည်ရွယ်သည်။ ၎င်းတွင် ပုံမှန်အားဖြင့် ၆ လတစ်ကြိမ် release အသစ်ထုတ်ဝေပြီး ၂ နှစ်တစ်ကြိမ် Long-term Support (LTS) release ထုတ်ဝေသည်။
2. Red Hat Family
ဤအုပ်စုသည် software များအတွက် rpm package format ကို အသုံးပြုသည်။
 * Red Hat Enterprise Linux (RHEL): ကုမ္ပဏီများအတွက် ခိုင်မာသော enterprise solution အဖြစ် ထုတ်လုပ်ထားခြင်းဖြစ်ပြီး အခကြေးငွေပေးဆောင်ရသော subscription များ ပါဝင်သည်။
 * CentOS: RHEL ၏ source code ကို အခြေခံကာ အခမဲ့ ထုတ်လုပ်ထားသော Distro ဖြစ်သည်။
 * Fedora: Desktop ကွန်ပျူတာများအတွက် ရည်ရွယ်ပြီး နည်းပညာအသစ်များကို လျင်မြန်စွာ စမ်းသပ်အသုံးပြုလေ့ရှိသော project ဖြစ်သည်။
3. SUSE Family
 * SUSE Linux Enterprise Server (SLES): လုပ်ငန်းခွင်သုံး Server များအတွက် ထုတ်လုပ်ထားခြင်း ဖြစ်သည်။
 * openSUSE: Developer များအတွက် လွတ်လပ်စွာ စမ်းသပ်အသုံးပြုနိုင်သော Distro ဖြစ်သည်။
Embedded Systems
Embedded Systems ဆိုသည်မှာ ပိုမိုကြီးမားသော စနစ်တစ်ခုအတွင်း၌ သတ်မှတ်ထားသော အလုပ်တစ်ခုတည်းကို လုပ်ဆောင်ရန် တည်ဆောက်ထားသည့် Hardware နှင့် Software ပေါင်းစပ်မှု ဖြစ်သည်။ ဥပမာအားဖြင့် ဆေးဘက်ဆိုင်ရာ ကိရိယာများ၊ မော်တော်ကားများနှင့် စစ်ဘက်ဆိုင်ရာ အသုံးအဆောင်များတွင် တွေ့ရသည်။
လူကြိုက်များသော Embedded Linux များ:
 * Android: Google မှ တီထွင်ထားသော mobile Operating System ဖြစ်သည်။ ၎င်းသည် Linux Kernel ကို အခြေခံကာ touchscreen ကိရိယာများ၊ TV နှင့် နာရီများတွင် အသုံးပြုရန် ပြုပြင်ထားခြင်း ဖြစ်သည်။
 * Raspbian: Raspberry Pi ကွန်ပျူတာငယ်လေးများတွင် အသုံးပြုရန် Debian အပေါ် အခြေခံကာ ထုတ်လုပ်ထားခြင်း ဖြစ်သည်။
Linux and the Cloud
Cloud computing ဆိုသည်မှာ ကွန်ပျူတာ အရင်းအမြစ် (resources) များကို public cloud provider များထံမှ ဝယ်ယူသုံးစွဲခြင်း သို့မဟုတ် private cloud တည်ဆောက်၍ သုံးစွဲခြင်းကို ဆိုလိုသည်။ ၂၀၁၇ ခုနှစ် အစီရင်ခံစာများအရ Public Cloud workload များ၏ ၉၀ ရာခိုင်နှုန်းတွင် Linux ကို အသုံးပြုထားသည်။
Amazon Web Services (AWS), Google Cloud Platform (GCP) နှင့် Microsoft Azure စသည့် cloud provider တိုင်းသည် Linux ကို အခြေခံသော virtual machines များကို Infrastructure as a Service (IaaS) အဖြစ် ပံ့ပိုးပေးထားကြသည်။
အနှစ်ချုပ် (Summary)
ဤသင်ခန်းစာတွင် သင်ယူခဲ့ရသည်များမှာ -
 * Linux ၏ သမိုင်းကြောင်းနှင့် Unix အပေါ် အခြေခံခဲ့မှု။
 * အဓိက Linux Distribution မိသားစုများ (Debian, Red Hat, SUSE) နှင့် ၎င်းတို့၏ ကွဲပြားချက်များ။
 * Embedded Systems များတွင် Linux ကို အသုံးပြုပုံ (ဥပမာ - Android, Raspberry Pi)။
 * Cloud computing တွင် Linux ၏ အခန်းကဏ္ဍ။
