### နိဒါန်း (Introduction)
Linux သည် လူကြိုက်အများဆုံး operating system များထဲမှ တစ်ခုဖြစ်သည်။ ၎င်းကို ၁၉၉၁ ခုနှစ်တွင် Linus Torvalds မှ စတင်ဖော်ဆောင်ခဲ့ခြင်းဖြစ်သည်။ ဤ operating system သည် ၁၉၇၀ ပြည့်လွန်နှစ်များအတွင်း AT&T ဓာတ်ခွဲခန်းမှ ဖော်ဆောင်ခဲ့သော Unix အမည်ရှိ အခြား operating system တစ်ခုမှ စိတ်ကူးစိတ်သန်း ရယူထားခြင်း ဖြစ်သည်။ ထိုစဉ်က Unix သည် ကွန်ပျူတာအသေးစားများအတွက် ရည်ရွယ်ခဲ့သော်လည်း ထိုခေတ်က "အသေးစား" ဟု ဆိုရာ၌ အဲယားကွန်းတပ်ဆင်ထားသော ခန်းမကျယ်ကြီး မလိုအပ်ဘဲ လူနှစ်ဦး မနိုင်သော စက်ပစ္စည်းများကို ဆိုလိုခြင်းဖြစ်သည်။

Linus သည် ကျောင်းသားဘဝတွင် x86 platform များပေါ်တွင် အသုံးပြုနိုင်သည့် Unix နှင့် တူညီသော operating system တစ်ခုကို စတင်ဖန်တီးခဲ့သည်။ Linux သည် Unix ၏ အခြေခံမူဝါဒများနှင့် စိတ်ကူးစိတ်သန်းများကို အသုံးပြုထားသော်လည်း ၎င်းတွင် Unix ၏ code များ လုံးဝပါဝင်ခြင်းမရှိဘဲ သီးခြားလွတ်လပ်သော ပရောဂျက်တစ်ခုသာဖြစ်သည်။ Linux ကို ကုမ္ပဏီတစ်ခုတည်းမှ ပံ့ပိုးထားခြင်းမျိုးမဟုတ်ဘဲ နိုင်ငံတကာမှ ပရိုဂရမ်မာများ စုပေါင်းထားသော အဖွဲ့အစည်းမှ ထိန်းသိမ်းလုပ်ဆောင်နေခြင်းဖြစ်သည်။ ၎င်းကို အခမဲ့ ရရှိနိုင်ပြီး မည်သူမဆို ကန့်သတ်ချက်မရှိဘဲ အသုံးပြုနိုင်သည်။

### Linux ဖြန့်ချိမှုပုံစံများ (Distributions)
Linux distribution (သို့မဟုတ် distro) ဆိုသည်မှာ Linux kernel နှင့် အသုံးပြုသူများအတွက် လိုအပ်သော application များကို စုစည်း၍ ကုမ္ပဏီတစ်ခု သို့မဟုတ် အသုံးပြုသူအဖွဲ့အစည်းတစ်ခုမှ ထိန်းသိမ်းထားသော အစုအဝေးတစ်ခုဖြစ်သည်။ Distribution တစ်ခု၏ ရည်မှန်းချက်မှာ သတ်မှတ်ထားသော အသုံးပြုမှုပုံစံ သို့မဟုတ် အသုံးပြုသူအုပ်စုအတွက် အကောင်းဆုံးဖြစ်အောင် ပြင်ဆင်ပေးရန်ဖြစ်သည်။

အဓိက Linux မျိုးနွယ်စုများမှာ-

* **Debian Family:** dpkg package manager ကို အသုံးပြုသည်။
    * **Debian GNU/Linux:** ၁၉၉၃ ခုနှစ်တွင် Ian Murdock မှ စတင်ခဲ့ပြီး ယနေ့တိုင် စေတနာ့ဝန်ထမ်း ထောင်ပေါင်းများစွာမှ ထိန်းသိမ်းနေသည်။ အလွန်စိတ်ချရသော operating system ဖြစ်ရန် ရည်ရွယ်သည်။
    * **Ubuntu:** ၂၀၀၄ ခုနှစ်တွင် Mark Shuttleworth မှ စတင်ခဲ့ပြီး အသုံးပြုရလွယ်ကူသော desktop environment ရရှိရန် ရည်ရွယ်သည်။ ၂ နှစ်လျှင် တစ်ကြိမ် Long Term Support (LTS) ဗားရှင်းများကို ထုတ်ပေးသည်။

* **Red Hat Family:** rpm package format ကို အသုံးပြုသည်။
    * **Red Hat Enterprise Linux (RHEL):** စီးပွားရေးလုပ်ငန်းသုံး ခိုင်မာသော solution အဖြစ် ထုတ်လုပ်ထားသည်။ IBM မှ ၂၀၁၉ တွင် ဝယ်ယူခဲ့သည်။
    * **CentOS:** RHEL ၏ source code ကို အသုံးပြု၍ အခမဲ့ ပြန်လည်တည်ဆောက်ထားသော distribution ဖြစ်သည်။
    * **Fedora:** နည်းပညာအသစ်များကို ရှေ့ပြေးစမ်းသပ်သည့် distribution ဖြစ်ပြီး RHEL အတွက် စမ်းသပ်ခန်းသဖွယ် အသုံးပြုသည်။

* **SUSE Family:**
    * **openSUSE:** SUSE Linux Enterprise (SLE) နှင့် ဆက်စပ်နေပြီး YaST configuration tool ကြောင့် လူသိများသည်။

### မြှုပ်နှံထားသော စနစ်များ (Embedded Systems)
Embedded system ဆိုသည်မှာ ပိုမိုကြီးမားသော စက်ကိရိယာတစ်ခုအတွင်း သီးသန့်လုပ်ဆောင်ချက်တစ်ခုအတွက် ထည့်သွင်းထားသော ကွန်ပျူတာစနစ်ဖြစ်သည်။
* **Raspberry Pi:** အလွန်သေးငယ်ပြီး ဈေးနှုန်းသက်သာသော ကွန်ပျူတာဖြစ်ကာ ပညာရေးနှင့် ဝါသနာရှင်များအတွက် ရည်ရွယ်သည်။ ၎င်းအတွက် အဓိက distribution မှာ **Raspbian** (ယခု Raspberry Pi OS) ဖြစ်သည်။
* **Android:** Linux kernel ကို အခြေခံထားသော လူကြိုက်အများဆုံး mobile operating system ဖြစ်သည်။

### ကလောက်ပေါ်ရှိ Linux (Linux and the Cloud)
Cloud computing တွင် Linux သည် အဓိကကျသော အခန်းကဏ္ဍမှ ပါဝင်သည်။
* **IaaS (Infrastructure as a Service):** အခြေခံအဆောက်အအုံကို ဝန်ဆောင်မှုအဖြစ် ပေးခြင်းဖြစ်သည်။ Amazon Web Services (AWS) ကဲ့သို့သော provider များသည် Linux ကို အခြေခံ၍ ဝန်ဆောင်မှုပေးကြသည်။

ဤသင်ခန်းစာတွင် သင်လေ့လာခဲ့ရသည်မှာ -
* Linux သည် ၁၉၉၁ ခုနှစ်တွင် Linus Torvalds မှ စတင်ခဲ့ပြီး Unix ၏ အခြေခံမူများကို အသုံးပြုထားကြောင်း။
* Linux distribution ဆိုသည်မှာ kernel နှင့် application များကို စုစည်းထားသော အစုအဝေးဖြစ်ကြောင်း။
