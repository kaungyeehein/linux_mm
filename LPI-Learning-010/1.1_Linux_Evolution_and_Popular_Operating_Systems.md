### နိဒါန်း (Introduction)

Linux သည် လူကြိုက်အများဆုံး operating system များထဲမှ တစ်ခုဖြစ်သည်။ ၎င်း၏ ဖွံ့ဖြိုးတိုးတက်မှုကို ၁၉၉၁ ခုနှစ်တွင် Linus Torvalds မှ စတင်ခဲ့သည်။ ဤ operating system သည် ၁၉၇၀ ပြည့်လွန်နှစ်များအတွင်း AT&T ဓာတ်ခွဲခန်းများမှ ထုတ်လုပ်ခဲ့သော အခြား operating system တစ်ခုဖြစ်သည့် Unix ကို အခြေခံ၍ စိတ်ကူးရရှိခဲ့ခြင်း ဖြစ်သည်။ Unix ကို ကွန်ပျူတာ အသေးစားများအတွက် ရည်ရွယ်၍ ဖန်တီးခဲ့သည်။ ထိုခေတ်အခါက "အသေးစား" ကွန်ပျူတာ ဆိုသည်မှာ လေအေးပေးစက်တပ်ဆင်ထားသော ခန်းမတစ်ခုလုံးစာ နေရာယူရန်မလိုဘဲ တန်ဖိုးအားဖြင့် ဒေါ်လာတစ်သန်းအောက်ရှိသော စက်များကို ဆိုလိုခြင်းဖြစ်သည်။ နောက်ပိုင်းတွင် လူနှစ်ဦး မနိုင်နင်းနိုင်သော စက်များကို အသေးစားကွန်ပျူတာဟု သတ်မှတ်ခဲ့ကြသည်။ ထိုအချိန်တွင် x86 platform ကို အခြေခံသည့် ရုံးသုံးကွန်ပျူတာများကဲ့သို့သော စက်များပေါ်တွင် သင့်တင့်သောဈေးနှုန်းဖြင့် အသုံးပြုနိုင်သည့် Unix system များ အလွယ်တကူ မရှိသေးပေ။ ထို့ကြောင့် ထိုစဉ်က ကျောင်းသားတစ်ဦးဖြစ်သော Linus သည် ဤ platform ပေါ်တွင် အသုံးပြုနိုင်မည့် Unix နှင့် ဆင်တူသော operating system တစ်ခုကို စတင်အကောင်အထည်ဖော်ခဲ့သည်။ Linux သည် Unix ၏ အခြေခံမူဝါဒများနှင့် စိတ်ကူးစိတ်သန်းများကို အဓိကထား အသုံးပြုထားသော်လည်း ၎င်းကိုယ်တိုင်သည် သီးခြားလွတ်လပ်သော project တစ်ခုဖြစ်သောကြောင့် Unix ၏ ကုဒ်များ (code) ပါဝင်ခြင်း မရှိပါ။ Linux ကို ကုမ္ပဏီတစ်ခုတည်းက ပိုင်ဆိုင်ခြင်းမျိုးမဟုတ်ဘဲ နိုင်ငံတကာမှ ပရိုဂရမ်မာများ စုစည်းထားသော အဖွဲ့အစည်း (community) မှ ပံ့ပိုးပေးထားခြင်းဖြစ်သည်။ ၎င်းကို အခမဲ့ ရရှိနိုင်သောကြောင့် မည်သူမဆို ကန့်သတ်ချက်မရှိဘဲ အသုံးပြုနိုင်သည်။

### Distributions
Linux ဖြန့်ချိမှုပုံစံများ (Distributions)

Linux kernel တစ်ခုတည်းသည် အသုံးပြုသူတစ်ဦးအတွက် အသုံးဝင်သော operating system တစ်ခုဖြစ်ရန် မလုံလောက်ပါ။ အသုံးပြုသူများအတွက် အဆင်သင့်အသုံးပြုနိုင်သော စနစ်တစ်ခုဖြစ်စေရန် kernel နှင့်အတူ software library များ၊ utility များ၊ ပရိုဂရမ်များနှင့် configuration ဖိုင်များကို စုစည်းထားရန် လိုအပ်ပါသည်။ ဤကဲ့သို့ စုစည်းထားသောအထုပ်ကို Linux distribution (သို့မဟုတ် အရပ်ခေါ် "distro") ဟု ခေါ်ဆိုပါသည်။

Linux distribution အမျိုးမျိုးရှိကြပြီး ၎င်းတို့သည် ရည်ရွယ်ချက်အမျိုးမျိုးပေါ်မူတည်၍ ကွဲပြားကြသည်။ အဓိကအားဖြင့် အောက်ပါအတိုင်း ခွဲခြားနိုင်သည်-

#### ၁။ အဓိက Distribution မိသားစုများ (Major Distribution Families)

* **Debian Family:** အခြေခံအကျဆုံးနှင့် လူကြိုက်အများဆုံး မိသားစုတစ်ခုဖြစ်သည်။
    * **Debian:** အလွန်တည်ငြိမ်ပြီး open source မူဝါဒကို တင်းတင်းကျပ်ကျပ် လိုက်နာသော distribution ဖြစ်သည်။
    * **Ubuntu:** Debian ကို အခြေခံထားပြီး အသုံးပြုရလွယ်ကူမှု (User-friendliness) ကို အဓိကထားသည်။ Desktop ရော Server အတွက်ပါ အသုံးများသည်။ **LTS (Long Term Support)** ဗားရှင်းများသည် ငါးနှစ်အထိ အထောက်အပံ့ပေးသည်။
    * **Linux Mint:** Ubuntu ကို အခြေခံထားပြီး Desktop အသုံးပြုသူများအတွက် ပိုမိုသက်သောင့်သက်သာရှိအောင် ပြုလုပ်ထားသည်။

* **Red Hat Family:** စီးပွားရေးလုပ်ငန်းသုံး (Enterprise) အတွက် အဓိကရည်ရွယ်သည်။
    * **Red Hat Enterprise Linux (RHEL):** ကုမ္ပဏီများအတွက် အခကြေးငွေဖြင့် အထောက်အပံ့ပေးသော distribution ဖြစ်သည်။
    * **CentOS:** RHEL ၏ source code ကို အခြေခံထားသော အခမဲ့ဗားရှင်းဖြစ်သည်။ (လက်ရှိတွင် CentOS Stream အဖြစ် ပြောင်းလဲသွားပြီဖြစ်သည်)။
    * **Scientific Linux:** သိပ္ပံနည်းကျ သုတေသနလုပ်ငန်းများအတွက် RHEL ကို အခြေခံ၍ ဖန်တီးထားသည်။

* **SUSE Family:**
    * **SUSE Linux Enterprise (SLE):** စီးပွားရေးလုပ်ငန်းသုံးအတွက် ရည်ရွယ်သည်။
    * **openSUSE:** လူတိုင်း အခမဲ့အသုံးပြုနိုင်သော ဗားရှင်းဖြစ်သည်။

#### ၂။ ရွေးချယ်ရာတွင် ထည့်သွင်းစဉ်းစားရမည့်အချက်များ
Distribution တစ်ခုကို ရွေးချယ်ရာတွင် အောက်ပါတို့ကို ကြည့်ရှုသင့်သည်-
* **Maintenance Cycle:** software update များကို မည်မျှကြာကြာ ပံ့ပိုးပေးသနည်း။
* **Software Availability:** မိမိလိုအပ်သော software များ အလွယ်တကူ ရနိုင်သလား။
* **Support:** အကူအညီလိုအပ်ပါက မည်သို့သော community သို့မဟုတ် commercial support ရနိုင်သနည်း။

### Embedded Systems
မြှုပ်နှံထားသော စနစ်များ (Embedded Systems)

Linux ကို ကွန်ပျူတာများနှင့် ဆာဗာများတွင်သာမက Embedded Systems ဟုခေါ်သော သီးခြားလုပ်ဆောင်ချက်တစ်ခုတည်းအတွက် ရည်ရွယ်ထုတ်လုပ်ထားသည့် စနစ်များတွင်လည်း ကျယ်ပြန့်စွာ အသုံးပြုကြသည်။

* **အဓိပ္ပာယ်သတ်မှတ်ချက်:** Embedded system ဆိုသည်မှာ ပိုမိုကြီးမားသော စက်ပစ္စည်း သို့မဟုတ် စနစ်တစ်ခုအတွင်း ကိန်းအောင်းနေပြီး သီးခြားလုပ်ဆောင်ချက်တစ်ခုကိုသာ လုပ်ဆောင်ပေးသည့် ကွန်ပျူတာစနစ်တစ်ခုဖြစ်သည်။
* **အိမ်သုံးနှင့် လူသုံးကုန်ပစ္စည်းများ:** ယနေ့ခေတ်တွင် အိမ်သုံးလျှပ်စစ်ပစ္စည်းများဖြစ်သော Smart TV များ၊ ရေခဲသေတ္တာများ၊ အဝတ်လျှော်စက်များနှင့် ကားဖျော်ဖြေရေးစနစ်များ (In-car entertainment systems) တွင် Linux ကို အခြေခံ၍ အသုံးပြုလာကြသည်။
* **မိုဘိုင်းလ်ဖုန်းများ (Android):** ကမ္ဘာပေါ်တွင် အသုံးအများဆုံး မိုဘိုင်းလ် operating system ဖြစ်သည့် Android သည် Linux kernel ကို အခြေခံထားခြင်းဖြစ်သည်။ ၎င်းသည် Linux ၏ စွမ်းဆောင်ရည်နှင့် လုံခြုံရေးကို မိုဘိုင်းလ်ဖုန်းများပေါ်တွင် အသုံးချထားခြင်းဖြစ်သည်။
* **Raspberry Pi:** ၎င်းသည် စာသင်ကျောင်းများနှင့် နည်းပညာဝါသနာရှင်များကြား ရေပန်းစားသော အိတ်ဆောင်ကွန်ပျူတာအသေးစားလေးဖြစ်သည်။ Raspberry Pi အတွက် အဓိကအသုံးပြုသော operating system မှာ Debian ကို အခြေခံထားသည့် **Raspbian (Raspberry Pi OS)** ဖြစ်သည်။
* **အားသာချက်:** Linux သည် ပေါ့ပါးခြင်း၊ ပြုပြင်ပြောင်းလဲရလွယ်ကူခြင်းနှင့် အခမဲ့ရရှိနိုင်ခြင်းတို့ကြောင့် embedded device ထုတ်လုပ်သူများအတွက် အသင့်တော်ဆုံး ရွေးချယ်မှုဖြစ်လာသည်။

### Linux and the Cloud
Linux နှင့် ကလောက် (Cloud)

Cloud computing နယ်ပယ်တွင် Linux သည် ကျောရိုးသဖွယ် အရေးပါသော အခန်းကဏ္ဍမှ ပါဝင်နေသည်။

* **Cloud ဆိုသည်မှာ:** Cloud computing ဆိုသည်မှာ မိမိ၏ ကိုယ်ပိုင်ကွန်ပျူတာတွင် software များ သိုလှောင်ခြင်း၊ အသုံးပြုခြင်းမျိုးမဟုတ်ဘဲ အင်တာနက်ပေါ်ရှိ အဝေးတစ်နေရာမှ ဆာဗာများ၏ စွမ်းဆောင်ရည် (Computing power) နှင့် သိုလှောင်မှု (Storage) ကို ငှားရမ်းအသုံးပြုခြင်းဖြစ်သည်။
* **အသုံးများသော နေရာများ:** Amazon Web Services (AWS), Google Cloud Platform (GCP) နှင့် Microsoft Azure ကဲ့သို့သော နာမည်ကြီး cloud ဝန်ဆောင်မှုပေးသူများ၏ ဆာဗာအများစုသည် Linux ပေါ်တွင်သာ အခြေခံ၍ လည်ပတ်နေကြသည်။
* **Virtualization:** Cloud ပေါ်တွင် virtual machine များစွာကို တစ်ပြိုင်နက်တည်း အသုံးပြုနိုင်ရန် Linux က ကူညီပေးသည်။ အသုံးပြုသူများသည် မိမိတို့ လိုအပ်သော Linux distribution (ဥပမာ - Ubuntu, CentOS သို့မဟုတ် Debian) ကို မိနစ်ပိုင်းအတွင်း cloud ပေါ်တွင် တပ်ဆင်အသုံးပြုနိုင်ကြသည်။
* **အကျိုးကျေးဇူး:** Cloud တွင် Linux ကို အသုံးပြုခြင်းဖြင့် ကုန်ကျစရိတ်သက်သာခြင်း၊ စနစ်ကို လိုသလို ချဲ့ထွင်နိုင်ခြင်း (Scalability) နှင့် စွမ်းဆောင်ရည် ကောင်းမွန်ခြင်း စသည့် အကျိုးကျေးဇူးများကို ရရှိစေသည်။

### Summary (အနှစ်ချုပ်)

ဤသင်ခန်းစာတွင် သင်ယူခဲ့ရသည်မှာ -
* **Linux ၏ မူလအစ:** Linux ကို ၁၉၉၁ ခုနှစ်တွင် Linus Torvalds မှ Unix ကို အခြေခံ၍ စတင်ဖန်တီးခဲ့သည်။
* **Distributions:** Linux kernel ကို အခြေခံ software များနှင့် စုစည်းထားသော distro အမျိုးမျိုး (ဥပမာ - Ubuntu, Fedora, Debian) ရှိသည်။
* **Embedded Systems:** Linux ကို ဖုန်းများ (Android)၊ အိမ်သုံးပစ္စည်းများနှင့် Raspberry Pi ကဲ့သို့သော စက်ပစ္စည်းအသေးစားများတွင် အသုံးပြုသည်။
* **Cloud Computing:** အင်တာနက်ပေါ်ရှိ cloud ဝန်ဆောင်မှုအများစုသည် ၎င်းတို့၏ စနစ်များကို လည်ပတ်ရန် Linux ကို အဓိက အသုံးပြုကြသည်။
* **Community:** Linux သည် ကုမ္ပဏီတစ်ခုတည်းပိုင်မဟုတ်ဘဲ ကမ္ဘာတစ်ဝှမ်းရှိ developer များ စုပေါင်းပံ့ပိုးထားသော Open Source project တစ်ခုဖြစ်သည်။

### Answers to Explorational Exercises
ရှာဖွေလေ့လာရန် လေ့ကျင့်ခန်းများအတွက် အဖြေများ

၁။ သင်၏ ကွန်ပျူတာ သို့မဟုတ် မိုဘိုင်းလ်ဖုန်းတွင် အသုံးပြုနေသော operating system အကြောင်း-
   * အကယ်၍ သင်သည် Android ဖုန်းကို အသုံးပြုနေပါက ၎င်းသည် Linux kernel ကို အခြေခံထားပါသည်။
   * အကယ်၍ သင်သည် Apple (iPhone သို့မဟုတ် Mac) ကို အသုံးပြုနေပါက ၎င်းသည် Unix-like ဖြစ်သော်လည်း Linux မဟုတ်ပါ။
   * Windows သည် Linux kernel ကို အခြေခံထားခြင်းမရှိသော်လည်း WSL (Windows Subsystem for Linux) မှတစ်ဆင့် Linux ကို အသုံးပြုနိုင်သည်။

၂။ Linux Distribution Timeline အကြောင်း-
   * Linux သည် Unix ၏ source code ကို အသုံးမပြုဘဲ သီးခြားလွတ်လပ်စွာ ဖန်တီးခဲ့သော်လည်း Unix ၏ အခြေခံဒီဇိုင်းနှင့် မူဝါဒများကို လိုက်နာထားသည့် "Unix-like" စနစ်ဖြစ်သည်။
   * Timeline ကို ကြည့်ရှုခြင်းဖြင့် Debian, Slackware နှင့် Red Hat ကဲ့သို့သော အစောပိုင်း distribution များမှတစ်ဆင့် ယနေ့ခေတ် ခေတ်စားနေသော Ubuntu နှင့် အခြား distro များ မည်သို့ဆင်းသက်လာသည်ကို မြင်တွေ့နိုင်သည်။
