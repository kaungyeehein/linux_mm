### နိဒါန်း (Introduction)
ဟာ့ဒ်ဝဲ (Hardware) မရှိလျှင် ဆော့ဖ်ဝဲ (Software) ဆိုသည်မှာ စာပေပုံစံတစ်မျိုးထက် မပိုပါ။ ဟာ့ဒ်ဝဲသည် ဆော့ဖ်ဝဲမှ ညွှန်ကြားထားသော အမိန့်များကို လုပ်ဆောင်ပေးပြီး သိမ်းဆည်းခြင်း (storage)၊ ထည့်သွင်းခြင်း (input) နှင့် ထုတ်ပေးခြင်း (output) အတွက် လုပ်ငန်းစဉ်များကို ပံ့ပိုးပေးသည်။ Cloud (ကလောက်) ဆိုသည်မှာပင် နောက်ကွယ်တွင် ဟာ့ဒ်ဝဲဖြင့် တည်ဆောက်ထားခြင်း ဖြစ်သည်။ Operating system တစ်ခုအနေဖြင့် Linux ၏ တာဝန်တစ်ခုမှာ ဆော့ဖ်ဝဲများ ဟာ့ဒ်ဝဲကို အသုံးပြုနိုင်ရန်အတွက် interface များ ပံ့ပိုးပေးရန် ဖြစ်သည်။


### ပါဝါပေးစနစ်များ (Power Supplies)
ပါဝါပေးစနစ် (Power Supply Unit - PSU) သည် ပင်မလျှပ်စစ်ဓာတ်အားလိုင်းမှ လျှပ်စစ်စီးကြောင်းကို ကွန်ပျူတာ၏ အတွင်းပိုင်း အစိတ်အပိုင်းများ အသုံးပြုနိုင်သော ဗို့အားနိမ့် တိုက်ရိုက်လျှပ်စီး (DC) အဖြစ် ပြောင်းလဲပေးသည်။

### မိခင်ဘုတ် (Motherboard)
မိခင်ဘုတ် (Motherboard) သည် ကွန်ပျူတာ၏ အဓိက ပတ်လမ်းဘုတ်ပြား (circuit board) ဖြစ်သည်။ ၎င်းသည် CPU၊ memory နှင့် အခြားသော hardware components များအားလုံးကို တစ်ခုနှင့်တစ်ခု ဆက်သွယ်ပေးသည့် ဗဟိုချက်ဖြစ်သည်။

### မှတ်ဉာဏ် (Memory)
ကွန်ပျူတာတွင် အဓိကအားဖြင့် memory အမျိုးအစား နှစ်မျိုးရှိသည် -
1. **Random Access Memory (RAM):** ၎င်းသည် ကွန်ပျူတာ အလုပ်လုပ်နေစဉ်အတွင်း ဒေတာများကို ခေတ္တသိမ်းဆည်းထားသည့် နေရာဖြစ်သည်။ ကွန်ပျူတာ ပိတ်လိုက်ပါက ၎င်းအတွင်းရှိ ဒေတာများ ပျက်ပြယ်သွားမည်ဖြစ်သည်။
2. **Read-Only Memory (ROM):** ၎င်းသည် ကွန်ပျူတာ စတင်ပွင့်လာရန် (boot တက်ရန်) လိုအပ်သော အခြေခံ ညွှန်ကြားချက်များကို အမြဲတမ်း သိမ်းဆည်းထားသည့် နေရာဖြစ်သည်။

### ပရိုဆက်ဆာများ (Processors)
Central Processing Unit (CPU) ကို ကွန်ပျူတာ၏ ဦးနှောက်ဟု ခေါ်ဆိုနိုင်သည်။ ၎င်းသည် ဆော့ဖ်ဝဲမှ ပေးပို့သော ညွှန်ကြားချက်များကို တွက်ချက်ခြင်းနှင့် အကောင်အထည်ဖော်ခြင်းတို့ကို လုပ်ဆောင်သည်။ ယနေ့ခေတ် CPU များတွင် processor core အများအပြား ပါဝင်လေ့ရှိပြီး တစ်ပြိုင်နက်တည်းတွင် အလုပ်များစွာကို ပိုမိုမြန်ဆန်စွာ လုပ်ဆောင်နိုင်သည်။

### သိမ်းဆည်းခြင်း (Storage)
ဒေတာများကို ရေရှည်သိမ်းဆည်းရန်အတွက် storage device များကို အသုံးပြုသည်။
* **Hard Disk Drives (HDD):** သံလိုက်ပြားများ အသုံးပြု၍ ဒေတာ သိမ်းဆည်းသည့် စနစ်ဖြစ်သည်။
* **Solid State Drives (SSD):** Flash memory ကို အသုံးပြု၍ ဒေတာ သိမ်းဆည်းသည်။ HDD ထက် ပိုမိုမြန်ဆန်ပြီး ခိုင်ခံ့သည်။

### အပိုင်းကန့်ခြင်း (Partitions)
Storage device တစ်ခုကို ပိုမိုထိရောက်စွာ စီမံခန့်ခွဲနိုင်ရန်အတွက် အပိုင်းငယ်များ (Partitions) ခွဲခြားနိုင်သည်။ ဥပမာအားဖြင့် Operating System အတွက် အပိုင်းတစ်ခုနှင့် အသုံးပြုသူ၏ ဒေတာများအတွက် အပိုင်းတစ်ခု ခွဲခြားထားနိုင်သည်။

### အနီးနားရှိ ဆက်စပ်ပစ္စည်းများ (Peripherals)
ကွန်ပျူတာ၏ အပြင်ဘက်မှ ဆက်သွယ်အသုံးပြုသော ပစ္စည်းများကို Peripherals ဟု ခေါ်သည်။ ဥပမာ - ကီးဘုတ်၊ မောက်စ်၊ မော်နီတာ၊ ပရင်တာ နှင့် ပြင်ပ storage များ (USB stick)။

### Drivers နှင့် Device Files (Drivers and Device Files)
Operating system သည် hardware တစ်ခုချင်းစီနှင့် ဆက်သွယ်နိုင်ရန် Driver ဟုခေါ်သော ဆော့ဖ်ဝဲငယ်လေးများ လိုအပ်သည်။ Linux တွင် hardware device တိုင်းကို `/dev/` directory အောက်ရှိ file များအဖြစ် ကိုယ်စားပြု ဖော်ပြလေ့ရှိသည်။ ဥပမာအားဖြင့် ပထမဆုံး hard drive ကို `/dev/sda` အဖြစ် တွေ့မြင်နိုင်သည်။
