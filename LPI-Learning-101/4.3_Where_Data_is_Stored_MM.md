# 4.3 Where Data is Stored
## (ဒေတာများကို ဘယ်မှာ သိမ်းဆည်းထားသလဲ)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux စနစ်အတွင်း ဒေတာများ၊ Configuration ဖိုင်များ၊ Process များနှင့် Memory အမျိုးအစားများကို ရှင်းပြထားပါတယ်။

---

## Programs နှင့် Configuration
- Program files – /bin, /usr/bin
- Configuration files – /etc

---

## Linux Kernel
- Linux OS ရဲ့ အဓိကအပိုင်း
- Hardware နှင့် Software ကြား အလုပ်လုပ်ပေး

---

## Hardware Devices
- Device files – /dev
- Block device နှင့် Character device

---

## Memory Types
- RAM – Primary memory
- Swap – Virtual memory

---

## Exam Focus – 4.3
- /etc = configuration
- /dev = devices
- Kernel = core OS
