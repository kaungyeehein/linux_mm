# 3.1 Archiving Files on the Command Line
## (Command Line ဖြင့် ဖိုင်များကို Archive နှင့် Compression လုပ်ခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux command line ကို အသုံးပြုပြီး ဖိုင်များကို စုစည်းခြင်း (archiving) နှင့် ဖိုင်အရွယ်အစား လျှော့ချခြင်း (compression) ကို ဘယ်လိုလုပ်ရမလဲ ဆိုတာကို ရှင်းပြထားပါတယ်။

---

## Archiving ဆိုတာဘာလဲ
- Archiving ဆိုတာ ဖိုင်များစွာကို **ဖိုင်တစ်ခုအဖြစ် စုစည်းခြင်း** ဖြစ်ပါတယ်
- Archiving သည် compression မလုပ်ဘဲ စုစည်းခြင်းသာ ဖြစ်နိုင်ပါတယ်

## tar Command
- Linux တွင် အများဆုံး အသုံးပြုတဲ့ archiving tool

### tar အခြေခံ option များ
- `c` – create (archive ဖန်တီး)
- `x` – extract (archive ဖြုတ်)
- `t` – list (အတွင်းပါဝင်မှုကြည့်)
- `v` – verbose (အသေးစိတ်ပြ)
- `f` – file name သတ်မှတ်

ဥပမာ:
```
tar -cvf backup.tar /home/user
```

---

## Compression Tools
- **gzip** – `.gz`
- **bzip2** – `.bz2`
- **xz** – `.xz`

ဥပမာ:
```
tar -czvf backup.tar.gz /home/user
```

---

## ZIP Files
- Windows နှင့် Compatible
- `zip`, `unzip` command များအသုံးပြု

---

## Exam Focus – 3.1
- tar = archive tool
- gzip/bzip2/xz = compression
- tar -czf = create + gzip
