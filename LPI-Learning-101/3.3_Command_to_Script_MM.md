# 3.3 Turning Commands into a Script
## (Command များကို Script အဖြစ် ပြောင်းလဲခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Command line command များကို စုပေါင်းပြီး Script ဖိုင်အဖြစ် ပြုလုပ်နည်းကို ရှင်းပြထားပါတယ်။

---

## Script ဆိုတာဘာလဲ
- Script ဆိုတာ command များကို ဖိုင်တစ်ခုထဲမှာ စုပေါင်းရေးထားခြင်း
- Automation လုပ်ရန် အသုံးပြု

---

## Shell Script အခြေခံ

### Shebang
```
#!/bin/bash
```
- Script ကို ဘယ် shell နဲ့ run မလဲ ဆိုတာ သတ်မှတ်

### Script ဖန်တီးခြင်း
1. File ဖန်တီး
2. Command များရေး
3. Execute permission ပေး

ဥပမာ:
```
chmod +x script.sh
./script.sh
```

---

## Variables
- Script ထဲတွင် ဒေတာ သိမ်းရန် အသုံးပြု

ဥပမာ:
```
name=Linux
echo $name
```

---

## Exit Code
- Command အောင်မြင်မှုကို ပြသ
- `0` = success
