# 1.3 Open Source Software and Licensing
## (Open Source Software နှင့် License အမျိုးအစားများ)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Open Source ဆိုတာဘာလဲ၊ Free Software အယူအဆ၊ License အမျိုးအစားများ (Copyleft / Permissive) နှင့် Open Source Business Model များကို ရှင်းပြထားပါတယ်။

---

## Open Source အဓိပ္ပါယ်
- Source code ကို ကြည့်နိုင်၊ ပြင်နိုင်၊ မျှဝေနိုင်
- Free ဆိုတာ ပိုက်ဆံမဟုတ်ဘဲ Freedom ကို ဆိုလို

---

## Free Software vs Open Source
- FSF – Freedom ကို ဦးစားပေး
- OSI – Business-friendly
- FLOSS = Free / Libre Open Source Software

---

## Four Freedoms
1. Program ကို run လုပ်နိုင်
2. Source code ကို လေ့လာ/ပြင်နိုင်
3. Software ကို မျှဝေနိုင်
4. ပြင်ထားသော version ကို မျှဝေနိုင်

---

## License Types

### Copyleft
- GPL, LGPL, AGPL
- ပြင်ပြီး မျှဝေမယ်ဆိုရင် Open Source ဖြစ်ရမယ်

### Permissive
- BSD License
- Closed source ထဲ ထည့်သုံးနိုင်

### Creative Commons
- Documentation, Learning material များအတွက်

---

## Open Source Business Models
- Dual licensing
- Support / Consulting
- SaaS

---

## Exam Focus
- GPL = Copyleft
- BSD = Permissive
- Free ≠ Free of charge
