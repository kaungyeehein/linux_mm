# 4.2 Understanding Computer Hardware
## (Computer Hardware ကို နားလည်ခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Computer Hardware အစိတ်အပိုင်းများ (CPU, Memory, Storage) နှင့် Linux က ဘယ်လိုဆက်ဆံအသုံးပြုသလဲ ဆိုတာကို ရှင်းပြထားပါတယ်။

---

## Hardware Components

### CPU (Processor)
- Command များကို လုပ်ဆောင်ပေးသော အဓိကအစိတ်အပိုင်း
- Architecture – x86, x64, ARM

### Memory (RAM)
- Running program များအတွက် အသုံးပြု
- Volatile memory

### Storage
- HDD, SSD
- Data သိမ်းဆည်းရန်

---

## Device Drivers
- Hardware ကို OS နှင့် Application များ အသုံးပြုနိုင်အောင် ချိတ်ဆက်ပေး
- Linux တွင် Driver များ အများအားဖြင့် Kernel ထဲတွင် ပါဝင်
