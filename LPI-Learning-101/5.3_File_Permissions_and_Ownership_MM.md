# 5.3 Managing File Permissions and Ownership
## (File Permission နှင့် Ownership စီမံခန့်ခွဲခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux File Permission (r, w, x) နှင့် Ownership (user, group) ကို နားလည်နိုင်အောင် ရှင်းပြထားပါတယ်။

---

## File Permission အယူအဆ
- **r (read)** – ဖတ်ခွင့်
- **w (write)** – ပြင်/ရေးခွင့်
- **x (execute)** – run လုပ်ခွင့်

## Permission အုပ်စုများ
- User (owner)
- Group
- Others

---

## Permission ကြည့်ရန်
```
ls -l
```

## Permission ပြောင်းရန်
- `chmod`
- Numeric mode (755, 644)

---

## Ownership ပြောင်းရန်
- `chown`
- `chgrp`

---

## Exam Focus – 5.3
- r w x
- chmod, chown
- user / group / others
