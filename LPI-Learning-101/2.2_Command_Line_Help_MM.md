# 2.2 Using the Command Line to Get Help
## (Command Line မှ အကူအညီရယူခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux command များကို ဘယ်လိုလေ့လာရမလဲ၊ Documentation ကို ဘယ်လိုဖတ်ရမလဲ ဆိုတာကို ရှင်းပြထားပါတယ်။

---

## Help Commands

### man
- Manual pages ဖတ်ရန်
```
man ls
```

### --help
- Command အကျဉ်းချုပ်
```
ls --help
```

### info
- GNU documentation
```
info ls
```

## Documentation Location
- `/usr/share/doc`

## File Searching
- `locate` – database အခြေခံ
- `find` – real-time search

---

## Exam Focus
- man = primary help
- --help = quick help
- locate vs find
