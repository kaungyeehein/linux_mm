# 2.3 Using Directories and Listing Files
## (Directory များနှင့် File များကို စီမံခန့်ခွဲခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux filesystem အခြေခံ၊ Directory navigation၊ File listing နည်းလမ်းများကို ရှင်းပြထားပါတယ်။

---

## Filesystem အခြေခံ
- `/` = root directory
- `/home` = user home

## Navigation Commands
- `pwd` – လက်ရှိနေရာ
- `cd` – directory ပြောင်း

## Listing Files
- `ls` – list files
- `ls -l` – long list
- `ls -a` – hidden files

## Path Types
- Absolute path – `/home/user`
- Relative path – `../Documents`

## Special Directories
- `.` – current
- `..` – parent
- `~` – home
