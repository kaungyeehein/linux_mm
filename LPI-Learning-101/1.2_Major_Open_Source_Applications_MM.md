# 1.2 Major Open Source Applications
## (အရေးကြီးသော Open Source Application များ)

---

## အပိုင်းအကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux စနစ်ပေါ်မှာ အသုံးပြုနိုင်တဲ့ အရေးကြီးသော **Open Source Application များ** ကို အပြည့်အစုံ လေ့လာနိုင်အောင် ရှင်းပြထားပါတယ်။ Office software၊ Web browser၊ Multimedia application၊ Server program၊ Data sharing tool နဲ့ Programming language များကို Beginner အဆင့်မှ စတင် နားလည်နိုင်အောင် ဖော်ပြထားပါတယ်။

---

## Application ဆိုတာဘာလဲ
- **Application** ဆိုတာ အသုံးပြုသူ (User) က တိုက်ရိုက်အသုံးပြုတဲ့ Software ဖြစ်ပါတယ်
- Linux Distribution များမှာ
  - Default application အချို့ ပါဝင်ပြီးသား
  - လိုအပ်ပါက ထပ်မံ install လုပ်နိုင်ပါတယ်

---

## Software Package Concept

### Package ဆိုတာဘာလဲ
- Linux မှာ Software တစ်ခုကို **Package** အနေနဲ့ စီမံခန့်ခွဲပါတယ်
- Package တစ်ခုထဲမှာ
  - Program (Executable)
  - Configuration files
  - Documentation
  ပါဝင်ပါတယ်

### Package Manager
- Package Manager ဆိုတာ Software များကို
  - Install
  - Update
  - Remove
  လုပ်ရန် အသုံးပြုတဲ့ Tool ဖြစ်ပါတယ်

#### Distribution အလိုက် Package Manager
- **Debian / Ubuntu** → `apt`, `dpkg`
- **Red Hat / CentOS / Fedora** → `rpm`, `yum`, `dnf`

## Office Applications (ရုံးသုံး Software)

### LibreOffice
- Microsoft Office ရဲ့ Open Source အစားထိုး
- Office suite တစ်ခုဖြစ်ပြီး အောက်ပါ Application များ ပါဝင်ပါတယ်

#### LibreOffice Components
- **Writer** – စာရေးသားခြင်း
- **Calc** – Spreadsheet (Excel အစားထိုး)
- **Impress** – Presentation (PowerPoint အစားထိုး)
- **Draw** – Diagram / Drawing
- **Math** – သင်္ချာ Formula
- **Base** – Database Application

## Package Installation & Removal (ဥပမာ)

### Debian / Ubuntu
```bash
sudo apt install package_name
sudo apt remove package_name
```

### Red Hat / CentOS / Fedora
```bash
sudo yum install package_name
sudo yum remove package_name
```
