### နိဒါန်း (Introduction)
Open source application များသည် Linux operating system ၏ အောင်မြင်မှုအတွက် အရေးပါသော အစိတ်အပိုင်းဖြစ်သည်။ Linux အသုံးပြုသူတစ်ဦးအနေဖြင့် မိမိ၏ လုပ်ငန်းဆောင်တာများကို လုပ်ဆောင်ရန်အတွက် မည်သည့် software များ ရှိသည်ကို သိရှိရန် လိုအပ်သည်။ ဤသင်ခန်းစာတွင် နေ့စဉ်အသုံးပြုနေသော ရုံးသုံး application များမှစ၍ server များတွင် အသုံးပြုသော program များအထိ အဓိကကျသော open source application များကို လေ့လာသွားမည် ဖြစ်သည်။

### Software Packages (Software အစုအဝေးများ)
Linux တွင် software များကို package များအဖြစ် စုစည်းထားလေ့ရှိသည်။ Software package တစ်ခုတွင် အောက်ပါတို့ ပါဝင်သည်-
* Software ကို အသုံးပြုနိုင်ရန် လိုအပ်သော binary file များ။
* Software ၏ လုပ်ဆောင်ချက်များကို ထိန်းချုပ်ပေးသော configuration file များ။
* အသုံးပြုပုံ လမ်းညွှန်ချက်များပါဝင်သော documentation များ။
* ထို software အလုပ်လုပ်ရန် အခြားလိုအပ်သော software (dependencies) များ၏ စာရင်း။

Package များကို စီမံခန့်ခွဲရန် Package Manager များကို အသုံးပြုကြသည်။ ၎င်းသည် software များကို အလွယ်တကူ ရှာဖွေခြင်း၊ ထည့်သွင်းခြင်း (install)၊ အဆင့်မြှင့်တင်ခြင်း (update) နှင့် ဖယ်ရှားခြင်း (remove) တို့ကို လုပ်ဆောင်ပေးသည်။

### Package Installation (Package များ ထည့်သွင်းခြင်း)
Distribution အလိုက် အသုံးပြုသော package format နှင့် package manager များ ကွဲပြားကြသည်-
* **Debian မျိုးနွယ်စု (Ubuntu, Mint):** `.deb` format ကို အသုံးပြုပြီး `apt` သို့မဟုတ် `apt-get` tool များကို အသုံးပြု၍ install လုပ်ကြသည်။
* **Red Hat မျိုးနွယ်စု (CentOS, Fedora):** `.rpm` format ကို အသုံးပြုပြီး `yum` သို့မဟုတ် `dnf` tool များကို အသုံးပြုကြသည်။

Software များကို များသောအားဖြင့် Repository ဟုခေါ်သော online software သိုလှောင်ရုံများမှ ရယူလေ့ရှိသည်။ ၎င်းသည် အသုံးပြုသူများအတွက် စိတ်ချရပြီး စစ်ဆေးပြီးသား software များကို တစ်နေရာတည်းတွင် ရရှိစေသည်။

### Package Removal (Package များ ဖယ်ရှားခြင်း)
Software တစ်ခုကို မလိုအပ်တော့သည့်အခါ package manager ကို အသုံးပြု၍ ပြန်လည်ဖယ်ရှားနိုင်သည်။ ဖယ်ရှားရာတွင် software ကိုသာ ဖယ်ရှားခြင်း (remove) သို့မဟုတ် software နှင့်အတူ ၎င်း၏ configuration များပါ တစ်ခါတည်း ဖယ်ရှားခြင်း (purge) ဟူ၍ ရွေးချယ်နိုင်သည်။

### Office Applications (ရုံးသုံး Application များ)
Linux တွင် Microsoft Office နှင့် အလားတူသော လုပ်ဆောင်ချက်များကို ပေးစွမ်းနိုင်သည့် အခမဲ့ရုံးသုံး software များရှိသည်-
* **LibreOffice:** ၎င်းသည် အသုံးအများဆုံး ရုံးသုံး software အစုအဝေးဖြစ်ပြီး စာစီစာရိုက် (Writer)၊ စာရင်းဇယား (Calc) နှင့် presentation (Impress) တို့ ပါဝင်သည်။
* **Apache OpenOffice:** LibreOffice ၏ အခြေခံဖြစ်သော အခြား software အစုအဝေးတစ်ခု ဖြစ်သည်။

ဤ application များသည် Microsoft Office format များဖြစ်သော `.docx`, `.xlsx` နှင့် `.pptx` ဖိုင်များကိုလည်း ဖွင့်နိုင်၊ ပြုပြင်နိုင်စွမ်း ရှိကြသည်။

### Web Browsers (ဝက်ဘ်ဘရောက်ဆာများ)

Linux တွင် အင်တာနက်အသုံးပြုရန်အတွက် စွမ်းဆောင်ရည်မြင့်မားသော open source ဘရောက်ဆာများစွာ ရှိသည်-
* **Firefox:** Mozilla မှ ထုတ်လုပ်ပြီး Linux distribution အများစုတွင် default browser အဖြစ် ပါဝင်လေ့ရှိသည်။ ၎င်းသည် လုံခြုံရေးနှင့် privacy ကို ဦးစားပေးသော browser တစ်ခုဖြစ်သည်။
* **Chromium:** Google Chrome ၏ အခြေခံဖြစ်သော open source စီမံကိန်းဖြစ်သည်။ ၎င်းသည် မြန်ဆန်ပြီး Chrome တွင် ရရှိနိုင်သော feature အများစုကို ရရှိနိုင်သည်။

### Multimedia (မာလ်တီမီဒီယာ)

ဗီဒီယိုကြည့်ခြင်း၊ သီချင်းနားထောင်ခြင်းနှင့် ပုံပြင်ဆင်ခြင်းတို့အတွက် အောက်ပါ application များကို အသုံးများကြသည်-
* **VLC Media Player:** မည်သည့် video format ကိုမဆို ဖွင့်နိုင်စွမ်းရှိသော လူကြိုက်အများဆုံး player ဖြစ်သည်။
* **GIMP (GNU Image Manipulation Program):** Adobe Photoshop နှင့် အလားတူသော လုပ်ဆောင်ချက်များပါဝင်သည့် ပုံပြင်ဆင်ရေး software ဖြစ်သည်။
* **Audacity:** အသံဖမ်းယူခြင်းနှင့် အသံဖိုင်များကို ပြုပြင်တည်းဖြတ်ခြင်း (audio editing) အတွက် အသုံးပြုသည်။

### Server Programs (ဆာဗာ ပရိုဂရမ်များ)

Linux သည် server လောကတွင် အသုံးအများဆုံးဖြစ်ပြီး အောက်ပါ server application များသည် အဓိကကျသည်-
* **Apache နှင့် Nginx:** ကမ္ဘာပေါ်ရှိ website အများစုကို လွှင့်တင်ပေးနေသော Web Server များ ဖြစ်ကြသည်။
* **MariaDB နှင့် MySQL:** အချက်အလက်များကို သိမ်းဆည်းရန် အသုံးပြုသော Database Server များ ဖြစ်ကြသည်။
* **Samba:** Linux နှင့် Windows ကွန်ပျူတာများအကြား ဖိုင်များကို ဝေမျှအသုံးပြုနိုင်ရန် (file sharing) ဆောင်ရွက်ပေးသည်။

### Programming Languages (ပရိုဂရမ်မင်း ဘာသာစကားများ)

Linux ပတ်ဝန်းကျင်သည် developer များအတွက် အလွန်ကောင်းမွန်ပြီး အောက်ပါ language များအတွက် ပံ့ပိုးမှုအပြည့်အဝရှိသည်-
* **C နှင့် C++:** Linux kernel နှင့် system program များကို ရေးသားရာတွင် အဓိကအသုံးပြုသည်။
* **Python:** ဖတ်ရလွယ်ကူပြီး automation၊ data science နှင့် web development တို့တွင် အလွန်လူကြိုက်များသည်။
* **PHP:** Dynamic website များ (ဥပမာ- WordPress) ဖန်တီးရာတွင် web server ဘက်ခြုံ၌ အသုံးပြုသည်။
* **Java:** လုပ်ငန်းသုံး software ကြီးများ (enterprise applications) ရေးသားရာတွင် အသုံးများသည်။

### လမ်းညွှန်လေ့ကျင့်ခန်းများ (Guided Exercises)

၁။ figlet command ကို အသုံးပြု၍ terminal ပေါ်တွင် စာသားများကို အကြီးကြီးမြင်ရအောင် ပြုလုပ်လိုသော်လည်း command not found ဖြစ်နေပါက DEB အခြေခံ distribution တစ်ခုတွင် မည်သည့် command ဖြင့် ရှာဖွေနိုင်သနည်း။
* **apt-cache search figlet** သို့မဟုတ် **apt search figlet**

၂။ DEB package တစ်ခုကို install လုပ်ရန်အတွက် မည်သည့် command ကို အသုံးပြုရသနည်း။
* **sudo apt-get install package_name** သို့မဟုတ် **sudo apt install package_name**

၃။ RPM အခြေခံစနစ်တစ်ခု (ဥပမာ- Fedora) တွင် cowsay command ကို install လုပ်ရန် မည်သို့ရိုက်ရမည်နည်း။
* **sudo yum install cowsay** သို့မဟုတ် **sudo dnf install cowsay**

၄။ Microsoft Office ဖိုင်များနှင့် ကိုက်ညီမှုရှိပြီး Linux distribution အများစုတွင် default အဖြစ်ပါဝင်သော office suite မှာ မည်သည်နည်း။
* **LibreOffice**

၅။ Linux နှင့် Windows စနစ်များအကြား ဖိုင်များနှင့် printer များကို ဝေမျှအသုံးပြုနိုင်ရန် (Data sharing) ကူညီပေးသော application မှာ မည်သည်နည်း။
* **Samba**

### အနှစ်ချုပ် (Summary)

ဤသင်ခန်းစာတွင် သင်လေ့လာခဲ့ရသည်မှာ -
* Application ဆိုသည်မှာ အသုံးပြုသူ၏ လုပ်ငန်းဆောင်တာများအတွက် ရည်ရွယ်ဖန်တီးထားသော program များဖြစ်ကြောင်း။
* Linux distribution များသည် software များကို repository များတွင် သိမ်းဆည်းထားပြီး package manager (apt, yum, dnf) များဖြင့် စီမံခန့်ခွဲကြောင်း။
* LibreOffice သည် အဓိက open source office suite ဖြစ်ပြီး ODF standard ကို အသုံးပြုကြောင်း။
* Firefox နှင့် Chromium ကဲ့သို့သော web browser များ၊ VLC နှင့် GIMP ကဲ့သို့သော multimedia tool များအကြောင်း။
* Apache, Nginx (Web Servers)၊ MariaDB, MySQL (Databases) နှင့် Python, Java, C (Programming Languages) စသည့် server နှင့် development ဆိုင်ရာ software များအကြောင်း။
