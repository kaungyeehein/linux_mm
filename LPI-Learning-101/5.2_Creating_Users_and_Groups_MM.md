# 5.2 Creating Users and Groups
## (User နှင့် Group များ ဖန်တီးခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux တွင် User နှင့် Group များကို ဘယ်လိုဖန်တီး၊ စီမံခန့်ခွဲရမလဲ ဆိုတာကို ရှင်းပြထားပါတယ်။

---

## User နှင့် Group အယူအဆ
- User = Linux အသုံးပြုသူ
- Group = User များ စုပေါင်းထားသော အုပ်စု

---

## အရေးကြီးသော ဖိုင်များ
- `/etc/passwd` – User အချက်အလက်
- `/etc/group` – Group အချက်အလက်
- `/etc/shadow` – Password (hashed)

---

## User Management Commands
- `useradd` – user ဖန်တီး
- `userdel` – user ဖျက်
- `passwd` – password ပြောင်း

## Group Management Commands
- `groupadd`
- `groupdel`

---

## Exam Focus – 5.2
- /etc/passwd = user info
- /etc/shadow = password
- useradd / groupadd
