# 3.2 Searching and Extracting Data from Files
## (ဖိုင်များထဲမှ ဒေတာရှာဖွေခြင်းနှင့် ထုတ်ယူခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ ဖိုင်အတွင်းရှိ ဒေတာများကို ရှာဖွေခြင်း၊ စစ်ထုတ်ခြင်း၊ ပြန်လည်အသုံးချခြင်းအတွက် အသုံးပြုတဲ့ command များကို ရှင်းပြထားပါတယ်။

---

## Input / Output Redirection
- Command output ကို ဖိုင်ထဲသို့ ပြောင်းပေးခြင်း

### Redirection Symbols
- `>` – output overwrite
- `>>` – output append
- `<` – input from file

ဥပမာ:
```
ls > filelist.txt
```

---

## Pipe (|)
- Command တစ်ခုရဲ့ output ကို နောက် command ရဲ့ input အဖြစ် အသုံးပြု

ဥပမာ:
```
ls | grep txt
```

---

## grep Command
- ဖိုင်ထဲမှ စာကြောင်းရှာရန်

### အသုံးများသော option များ
- `-i` – case insensitive
- `-r` – recursive
- `-n` – line number ပြ

ဥပမာ:
```
grep -i error logfile.txt
```
