# 1.1 Linux Evolution and Popular Operating Systems
## (Linux ၏ ဖွံ့ဖြိုးတိုးတက်မှုနှင့် လူသုံးများသော Operating Systems)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux ဘယ်လိုစတင်ဖြစ်ပေါ်လာသလဲ၊ Unix နဲ့ ဆက်စပ်ပုံ၊ Linux Distribution ဆိုတာဘာလဲ၊ လူသုံးများတဲ့ Distribution များ၊ Embedded Systems နှင့် Cloud တွင် Linux ၏ အရေးပါမှုကို အစပြုသူနားလည်လွယ်အောင် ရှင်းပြထားပါတယ်။

---

## Linux Evolution
- Linux ကို **1991 ခုနှစ်**တွင် **Linus Torvalds** က စတင်ဖန်တီးခဲ့သည်
- Unix မှ အယူအဆများကို ယူထားသော်လည်း Unix code မသုံး
- အခမဲ့၊ PC (x86) ပေါ်တွင် အသုံးပြုနိုင်စေရန် ရည်ရွယ်
- Community-driven project ဖြစ်ပြီး ကုမ္ပဏီတစ်ခုတည်းမပိုင်

---

## Linux Distribution
- Distribution = Linux Kernel + Software + Tools
- Desktop / Server / Embedded အလိုက် ခွဲခြားထား

### Debian Family
- Debian GNU/Linux – Stable, Server များတွင် အသုံးများ
- Ubuntu – Beginner-friendly, LTS ပါရှိ

### Red Hat Family
- RHEL – Enterprise, Commercial support
- CentOS – Free RHEL
- Fedora – New technology test-bed

### SUSE Family
- SUSE / openSUSE – YaST tool ပါရှိ

### Independent Distributions
- Kali Linux – Security
- QubesOS – High security

---

## Embedded Systems
- Hardware + Software ပေါင်းစပ်ထားသော စနစ်
- Android (Linux kernel based)
- Raspberry Pi / Raspbian

---

## Linux and the Cloud
- Cloud တွင် Linux ကို အများဆုံး အသုံးပြု
- IaaS တွင် Linux Virtual Machine အဖြစ် run

---

## Exam Focus
- Linux = Open Source, Community-driven
- Distribution = Kernel + Software
- Ubuntu = Beginner-friendly
- RHEL = Enterprise
- Android = Linux kernel based
