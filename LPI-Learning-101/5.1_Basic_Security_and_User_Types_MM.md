# 5.1 Basic Security and Identifying User Types
## (အခြေခံလုံခြုံရေးနှင့် User အမျိုးအစားများ)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux စနစ်အတွက် အခြေခံလုံခြုံရေးအယူအဆများ၊ User အမျိုးအစားများ (root, normal user, system user) နှင့် Access Control ကို နားလည်နိုင်အောင် ရှင်းပြထားပါတယ်။

---

## User အမျိုးအစားများ
- **root user**
  - UID = 0
  - စနစ်အပေါ် အပြည့်အဝ အခွင့်အရေးရှိ
- **Normal user**
  - ပုံမှန်အသုံးပြုသူ
  - ကန့်သတ်ထားသောအခွင့်အရေးများသာရှိ
- **System user / Service user**
  - Service များအတွက် အသုံးပြု
  - Login မလုပ်နိုင်တတ်

---

## Security အခြေခံအယူအဆ
- Linux သည် Multi-user system ဖြစ်သည်
- User တစ်ယောက်စီကို သီးခြားခွဲခြားထား
- Permission နှင့် Authentication ဖြင့် လုံခြုံရေးထိန်းချုပ်

---
