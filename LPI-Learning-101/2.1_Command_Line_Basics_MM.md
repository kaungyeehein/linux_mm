# 2.1 Command Line Basics
## (Command Line အခြေခံများ)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Linux Command Line (Shell) အခြေခံများ၊ Command structure၊ Option နှင့် Argument များ၊ Quoting အခြေခံကို အစပြုသူနားလည်လွယ်အောင် ရှင်းပြထားပါတယ်။

---

## Shell ဆိုတာဘာလဲ
- Shell ဆိုတာ User နဲ့ Operating System ကြား ဆက်သွယ်ပေးတဲ့ Program
- Linux တွင် အများဆုံး အသုံးပြုတဲ့ Shell = **bash**

## Command Prompt
- `$` = normal user
- `#` = root user

## Command Structure
```
command [options] [arguments]
```
- command – run လုပ်မယ့် program
- options – behavior ပြောင်း
- arguments – target (file, directory)

ဥပမာ:
```
ls -l /home
```

## Command Types
- **Internal commands** – shell ထဲမှာပါ
- **External commands** – executable file

## Quoting
- Double quotes " " – variable အလုပ်လုပ်
- Single quotes ' ' – literal text
- Escape `\` – special meaning ဖယ်

---

## Exam Focus
- bash = default shell
- command = options + arguments
- single vs double quotes ခြားနားချက်
