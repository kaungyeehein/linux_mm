# 2.4 Creating, Moving and Deleting Files
## (File နှင့် Directory ဖန်တီး၊ ရွှေ့၊ ဖျက်ခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ File နှင့် Directory များကို ဖန်တီးခြင်း၊ ရွှေ့ခြင်း၊ ဖျက်ခြင်းအတွက် အသုံးပြုတဲ့ command များကို ရှင်းပြထားပါတယ်။

---

## Creating
- `touch` – file ဖန်တီး
- `mkdir` – directory ဖန်တီး

## Moving / Renaming
- `mv` – file/directory ရွှေ့ သို့မဟုတ် အမည်ပြောင်း

## Copying
- `cp` – file copy
- `cp -r` – directory copy

## Deleting
- `rm` – file ဖျက်
- `rm -r` – directory ဖျက်

## Globbing
- `*` – any characters
- `?` – single character

---

## Exam Focus
- rm -r သတိထား
- mv = move + rename
- globbing symbols
