# 4.1 Choosing an Operating System
## (Operating System ကို ရွေးချယ်ခြင်း)

---

## အကျဉ်းချုပ်
ဒီအပိုင်းမှာ Operating System (OS) ဆိုတာဘာလဲ၊ Linux Distribution ကို ဘယ်လိုရွေးချယ်ရမလဲ၊ Linux မဟုတ်တဲ့ OS များ (Windows, macOS) နဲ့ အခြေခံကွာခြားချက်များကို ရှင်းပြထားပါတယ်။

---

## Operating System ဆိုတာဘာလဲ
- Operating System ဆိုတာ Hardware နှင့် User / Application ကြား ဆက်သွယ်ပေးတဲ့ အခြေခံ Software ဖြစ်ပါတယ်
- Hardware resources (CPU, Memory, Storage) ကို စီမံခန့်ခွဲ

---

## Linux Distribution ရွေးချယ်ရာတွင် စဉ်းစားရမည့်အချက်များ
- အသုံးပြုမည့်ရည်ရွယ်ချက် (Desktop / Server)
- Stability
- Support period
- Hardware compatibility
- Community / Vendor support

---

## Non-Linux Operating Systems
- Windows – Proprietary, Desktop အသုံးများ
- macOS – Unix-based, Apple hardware အတွက်

---

## Exam Focus – 4.1
- OS = Hardware + Software interface
- Linux Distribution ရွေးချယ်မှု
