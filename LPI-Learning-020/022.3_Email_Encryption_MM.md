# 022.3 Email Encryption
## (Email လုံခြုံရေး: OpenPGP နှင့် S/MIME)

---

## အဓိကအယူအဆ
- Email ကို encrypt/sign လုပ်တာက Confidentiality/Integrity/Authenticity ကို ပံ့ပိုး
- Public key (asymmetric) အခြေခံ

---

## Digital Signatures
- Message digest (hash) ကို sender private key နဲ့ sign
- Receiver က sender public key နဲ့ verify
- Integrity + Non-repudiation ကို ပံ့ပိုး

---

## OpenPGP (PGP)
- Decentralized trust model
- User က key pair ကို ကိုယ်တိုင် generate/ manage
- Tool: **GnuPG (gpg)**
- Key servers မှ public keys ကို upload/download လုပ်နိုင်

### Key Server Workflow (အလွယ်)
1. User A key pair generate
2. Private key ကို လျှို့ဝှက်ထား
3. Public key ကို keyserver သို့ upload
4. User B က keyserver မှ download/import

---

## S/MIME
- Centralized trust model
- **CA-issued X.509 certificates** အခြေခံ
- Email clients အများစု support
- Confidentiality + Integrity + Authenticity အားလုံးပံ့ပိုး

---

## OpenPGP vs S/MIME (မှတ်သားရန်)
- OpenPGP: key servers, user-managed trust
- S/MIME: CA certificates, centralized trust
- နှစ်မျိုးလုံးမှာ public/private keys အသုံးပြု

---

## Thunderbird (Practical)
- OpenPGP key pair create/import
- S/MIME certificate import
- Recipient public key/certificate မရှိရင် encryption မလုပ်နိုင်
