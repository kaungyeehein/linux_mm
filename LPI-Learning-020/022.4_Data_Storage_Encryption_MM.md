# 022.4 Data Storage Encryption
## (ဒေတာသိုလှောင်မှု လျှို့ဝှက်ရေး)

---

## အကျဉ်းချုပ်
Data ကို စက်ထဲမှာ သိမ်းတဲ့အခါ device theft/loss အန္တရာယ်ရှိလို့ encryption လိုအပ်တတ်သည်။ ဒီအပိုင်းမှာ file/container encryption, full-disk encryption, cloud file encryption အခြေခံများကို ရှင်းပြထားပါတယ်။

---

## Encryption အမျိုးအစား
- **File encryption**: ဖိုင်တစ်ချို့ကိုသာ encrypt
- **Container encryption**: encrypted vault/container ထဲမှာ သိမ်း
- **Full-Disk Encryption (FDE)**: drive တစ်ခုလုံး encrypt (OS ပါ)

---

## Tools (Concept)
- **VeraCrypt**
  - Container သို့ full disk encryption
  - Strong passphrase/keyfile အရေးကြီး
- **Cryptomator**
  - Cloud storage သို့ upload မတင်ခင် client-side encrypt
  - Provider မဖတ်နိုင်အောင် ကာကွယ်
- **BitLocker (Windows)**
  - OS built-in FDE
  - TPM နဲ့ integration လုပ်နိုင်

---

## Good Practices
- Strong password + MFA (သို့) keyfile
- Backup ကိုလည်း encrypt
- Recovery keys ကို secure storage ထဲမှာ သိမ်း

---

## Exam Focus
- File vs Container vs FDE
- Client-side cloud encryption idea
- Key management အရေးကြီး
