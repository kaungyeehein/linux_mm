# 023.2 Application Security
## (အပ္ပလီကေးရှင်း လုံခြုံရေး)

---

## အဓိကအယူအဆ
- Software updates မလုပ်ရင် vulnerabilities နဲ့ malware ထိခိုက်လွယ်
- Trusted sources မှသာ software ထည့်သွင်းရမယ်

---

## Software Updates
- OS updates / application updates / firmware updates
- Patch management သည် risk reduction အတွက် အရေးကြီး

---

## Secure Procurement & Installation
- Official app stores / trusted repositories
- Signature/hash verify (concept)
- Pirated/cracked software မသုံး

---

## Common Vulnerabilities (Concept)
- Input validation flaws (SQL injection, command injection)
- Buffer overflow (legacy)
- Insecure defaults / weak auth

---

## Local Protective Software
- Host firewall
- Anti-malware
- Application firewall/WAF (concept)

---

## Exam Focus
- Update importance
- trusted sources
- common vulnerability types

---
