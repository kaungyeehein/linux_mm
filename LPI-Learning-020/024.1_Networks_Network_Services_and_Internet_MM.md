# 024.1 Networks, Network Services and the Internet
## (ကွန်ယက်အခြေခံများနှင့် Internet)

---

## Network Media & Devices
- Wired: Ethernet, fiber
- Wireless: Wi‑Fi
- Devices:
  - Switch (Layer 2, MAC)
  - Router (Layer 3, IP)
  - Access Point (Wi‑Fi)

---

## IP Networks & Internet
- **IP address**: device identity on network
- **IPv4**: limited addresses (NAT used)
- **IPv6**: huge address space

---

## Routing & ISP
- Router က packet ကို next-hop သို့ forward
- Default gateway က local network → internet ကို ချိတ်
- ISP က internet access ပေး

---

## Exam Focus
- Switch vs Router
- IPv4 vs IPv6 concept
- Default gateway role
